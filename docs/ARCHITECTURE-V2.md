## 1. 执行摘要

### 1.1 项目概述

SmartLink V2是一个面向企业的智能Agent管理平台，采用分层架构设计，旨在解决复杂业务场景下的智能代理编排问题。平台核心设计思想是"规划-执行分离"，通过PlanAgent进行意图理解和任务分解，SubAgent执行具体任务，实现高效、可扩展的AI工作流。

### 1.2 设计目标

| 目标维度 | 具体指标 |
|---------|---------|
| 性能 | 支持1,000+并发WebSocket连接，P99延迟<500ms |
| 可扩展性 | 水平扩展支持，无状态设计 |
| 可靠性 | 99.9%可用性，自动故障转移 |
| 安全性 | OAuth2认证，细粒度权限控制，审计日志 |
| 多租户 | 共享数据库+tenant_id隔离 |

### 1.3 技术栈选型

```
+-------------------------------------------------------------+
|                     技术栈概览                               |
+-------------------------------------------------------------+
|                                                              |
|  后端框架     |  Python + FastAPI + AgentScope              |
|  数据存储     |  SQLite(当前) / PostgreSQL(预留)             |
|  缓存/队列    |  SQLite(当前) / Redis(预留)                  |
|  LLM集成     |  LiteLLM (多厂商统一接口)                     |
|  消息总线     |  MessageHub (内部实现，预留Redis)            |
|  监控告警     |  Prometheus + Grafana                        |
|  容器化      |  Docker + Docker Compose                     |
+-------------------------------------------------------------+
```

### 1.4 核心调用链

```
用户请求 → 接入层 → PlanAgent(意图路由) → SubAgent(角色执行) → Skill/MCP → LLM响应
```

### 1.5 关键特性

- **智能路由**: PlanAgent自动理解用户意图并路由到合适的SubAgent
- **角色化执行**: SubAgent具备特定角色和职责，专注完成细分领域任务
- **插件化技能**: Skill系统支持热插拔，MCP协议兼容Anthropic生态
- **统一消息**: MessageHub实现Agent间异步通信和解耦
- **持久化记忆**: 多层级记忆系统，支持上下文连贯对话

## 3. 核心概念

### 3.1 PlanAgent (规划Agent)

PlanAgent是系统的"大脑"，负责理解用户意图并规划执行路径。

#### 3.1.1 职责定义

```python
class PlanAgent:
    """
    PlanAgent - 规划与路由Agent
    
    核心职责:
    1. 意图识别: 理解用户输入的真实意图
    2. 任务分解: 将复杂任务拆分为可执行的子任务
    3. 路由决策: 选择合适的SubAgent执行任务
    4. 结果聚合: 整合多个SubAgent的执行结果
    """
    
    async def process(self, user_input: str, context: Context) -> ExecutionPlan:
        # 1. 意图识别
        intent = await self.recognize_intent(user_input)
        
        # 2. 任务分解
        tasks = await self.decompose_task(intent, context)
        
        # 3. 路由决策
        assignments = await self.route_tasks(tasks)
        
        # 4. 构建执行计划
        return ExecutionPlan(tasks=tasks, assignments=assignments)
```


### 3.2 SubAgent (执行Agent)

SubAgent是系统的"执行者"，每个SubAgent具有特定的角色和能力。

#### 3.2.1 角色定义

```python
class SubAgent:
    """
    SubAgent - 执行Agent基类
    
    属性:
    - role: Agent角色标识
    - capabilities: 能力清单
    - tools: 可用工具集
    - memory: 记忆上下文
    """
    
    def __init__(self, role: str, capabilities: List[str]):
        self.role = role
        self.capabilities = capabilities
        self.tools: List[Tool] = []
        self.memory = AgentMemory()
    
    async def execute(self, task: Task, context: Context) -> TaskResult:
        """执行分配的任务"""
        # 1. 加载上下文记忆
        await self.memory.load(context.session_id)
        
        # 2. 选择合适的工具
        tools = self.select_tools(task)
        
        # 3. 调用LLM执行
        result = await self.llm_execute(task, tools)
        
        # 4. 保存执行记忆
        await self.memory.save(result)
        
        return result
```

#### 3.4.2 核心接口

```python
# agent/core/message_hub.py

from typing import Callable, Dict, Any, Optional
from enum import Enum
import asyncio
from datetime import datetime

class MessageType(Enum):
    """消息类型枚举"""
    TASK_ASSIGNMENT = "task_assignment"      # 任务分派
    TASK_RESULT = "task_result"              # 任务结果
    AGENT_HEARTBEAT = "agent_heartbeat"      # 心跳
    CONTEXT_SYNC = "context_sync"            # 上下文同步
    SYSTEM_EVENT = "system_event"            # 系统事件

class Message:
    """消息对象"""
    def __init__(
        self,
        msg_type: MessageType,
        sender: str,
        recipient: Optional[str],
        payload: Dict[str, Any],
        correlation_id: Optional[str] = None
    ):
        self.id = generate_uuid()
        self.type = msg_type
        self.sender = sender
        self.recipient = recipient            # None表示广播
        self.payload = payload
        self.correlation_id = correlation_id  # 用于关联请求-响应
        self.timestamp = datetime.utcnow()
        self.priority = 0                     # 优先级: 0=普通, >0=高优先级

class MessageHub:
    """
    消息总线 - Agent间通信中枢
    
    功能:
    - 发布/订阅模式
    - 点对点通信
    - 消息持久化
    - 负载均衡(预留)
    """
    
    def __init__(self, storage_backend: str = "memory"):
        self.storage = self._init_storage(storage_backend)
        self.subscribers: Dict[str, List[Callable]] = {}
        self.running = False
    
    async def publish(
        self, 
        message: Message, 
        persistent: bool = False
    ) -> bool:
        """
        发布消息
        
        Args:
            message: 消息对象
            persistent: 是否持久化
            
        Returns:
            bool: 发布成功/失败
        """
        # 1. 如需要，持久化消息
        if persistent:
            await self.storage.save(message)
        
        # 2. 路由消息
        if message.recipient:
            # 点对点
            await self._send_to_recipient(message)
        else:
            # 广播
            await self._broadcast(message)
        
        return True
    
    async def subscribe(
        self,
        msg_type: MessageType,
        handler: Callable[[Message], None],
        subscriber_id: str
    ) -> None:
        """订阅特定类型的消息"""
        key = f"{msg_type.value}:{subscriber_id}"
        if key not in self.subscribers:
            self.subscribers[key] = []
        self.subscribers[key].append(handler)
    
    async def request_response(
        self,
        request: Message,
        timeout: float = 30.0
    ) -> Optional[Message]:
        """
        请求-响应模式
        
        用于同步等待响应
        """
        future = asyncio.Future()
        
        # 注册等待响应
        correlation_id = request.correlation_id or generate_uuid()
        self._pending_responses[correlation_id] = future
        
        # 发送请求
        await self.publish(request)
        
        try:
            # 等待响应
            response = await asyncio.wait_for(future, timeout)
            return response
        except asyncio.TimeoutError:
            return None
```

```javascript
class SmartLinkWebSocket {
    constructor(url, options = {}) {
        this.url = url;
        this.reconnectInterval = options.reconnectInterval || 3000;
        this.maxReconnects = options.maxReconnects || 5;
        this.reconnectCount = 0;
        this.listeners = {};
        this.connect();
    }
    
    connect() {
        this.ws = new WebSocket(this.url);
        
        this.ws.onopen = () => {
            console.log('WebSocket connected');
            this.reconnectCount = 0;
            this.emit('connected');
            
            // 发送初始化消息
            this.send({
                type: 'init',
                data: { app_id: this.appId }
            });
        };
        
        this.ws.onmessage = (event) => {
            const message = JSON.parse(event.data);
            this.handleMessage(message);
        };
        
        this.ws.onerror = (error) => {
            console.error('WebSocket error:', error);
            this.emit('error', error);
        };
        
        this.ws.onclose = () => {
            console.log('WebSocket closed');
            this.emit('disconnected');
            
            // 自动重连
            if (this.reconnectCount < this.maxReconnects) {
                setTimeout(() => {
                    this.reconnectCount++;
                    this.connect();
                }, this.reconnectInterval);
            }
        };
    }
    
    handleMessage(message) {
        switch (message.type) {
            case 'token':
                this.emit('token', message.data);
                break;
            case 'tool_call':
                this.emit('toolCall', message.data);
                break;
            case 'tool_result':
                this.emit('toolResult', message.data);
                break;
            case 'complete':
                this.emit('complete', message.data);
                break;
            case 'error':
                this.emit('error', message.data);
                break;
            case 'ping':
                this.send({ type: 'pong' });
                break;
        }
    }
    
    send(data) {
        if (this.ws.readyState === WebSocket.OPEN) {
            this.ws.send(JSON.stringify(data));
        }
    }
    
    chat(message, context = {}) {
        this.send({
            type: 'chat',
            data: { message, context },
            timestamp: Date.now()
        });
    }
    
    on(event, callback) {
        if (!this.listeners[event]) {
            this.listeners[event] = [];
        }
        this.listeners[event].push(callback);
    }
    
    emit(event, data) {
        if (this.listeners[event]) {
            this.listeners[event].forEach(cb => cb(data));
        }
    }
    
    close() {
        this.ws.close();
    }
}
```

#### 4.4.2 Circuit Breaker (熔断器) 实现

```python
# core/circuit_breaker.py

import time
from enum import Enum
from typing import Callable, Optional
import asyncio

class CircuitState(Enum):
    CLOSED = "closed"      # 正常，允许请求
    OPEN = "open"          # 熔断，拒绝请求
    HALF_OPEN = "half_open"  # 半开，试探请求

class CircuitBreaker:
    """
    熔断器模式实现
    
    防止级联故障，保护系统稳定性
    """
    
    def __init__(
        self,
        name: str,
        failure_threshold: int = 5,      # 失败阈值
        recovery_timeout: float = 30.0,  # 恢复超时(秒)
        half_open_max_calls: int = 3     # 半开状态最大试探请求
    ):
        self.name = name
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.half_open_max_calls = half_open_max_calls
        
        self.state = CircuitState.CLOSED
        self.failure_count = 0
        self.last_failure_time: Optional[float] = None
        self.half_open_calls = 0
        self._lock = asyncio.Lock()
    
    async def call(self, func: Callable, *args, **kwargs):
        """执行受保护的调用"""
        async with self._lock:
            await self._update_state()
            
            if self.state == CircuitState.OPEN:
                raise CircuitBreakerOpenError(
                    f"Circuit breaker '{self.name}' is OPEN"
                )
            
            if self.state == CircuitState.HALF_OPEN:
                if self.half_open_calls >= self.half_open_max_calls:
                    raise CircuitBreakerOpenError(
                        f"Circuit breaker '{self.name}' half-open limit reached"
                    )
                self.half_open_calls += 1
        
        try:
            result = await func(*args, **kwargs)
            await self._on_success()
            return result
        except Exception as e:
            await self._on_failure()
            raise
    
    async def _update_state(self):
        """更新熔断器状态"""
        if self.state == CircuitState.OPEN:
            if self.last_failure_time:
                elapsed = time.time() - self.last_failure_time
                if elapsed >= self.recovery_timeout:
                    self.state = CircuitState.HALF_OPEN
                    self.half_open_calls = 0
                    print(f"Circuit '{self.name}' moved to HALF_OPEN")
    
    async def _on_success(self):
        """处理成功"""
        async with self._lock:
            if self.state == CircuitState.HALF_OPEN:
                self.state = CircuitState.CLOSED
                self.failure_count = 0
                self.half_open_calls = 0
                print(f"Circuit '{self.name}' CLOSED")
            else:
                self.failure_count = 0
    
    async def _on_failure(self):
        """处理失败"""
        async with self._lock:
            self.failure_count += 1
            self.last_failure_time = time.time()
            
            if self.state == CircuitState.HALF_OPEN:
                self.state = CircuitState.OPEN
                print(f"Circuit '{self.name}' OPENED (half-open failure)")
            elif self.failure_count >= self.failure_threshold:
                self.state = CircuitState.OPEN
                print(f"Circuit '{self.name}' OPENED (threshold reached)")
```

```python
from typing import Callable, List, Any

class FallbackStrategy:
    """降级策略管理器"""
    
    def __init__(self):
        self.fallbacks: Dict[str, List[Callable]] = {}
    
    def register(self, primary: str, fallbacks: List[Callable]):
        """注册降级链"""
        self.fallbacks[primary] = fallbacks
    
    async def execute(self, primary_func: Callable, *args, **kwargs) -> Any:
        """执行带降级的调用"""
        primary_name = primary_func.__name__
        chain = [primary_func] + self.fallbacks.get(primary_name, [])
        
        last_error = None
        for func in chain:
            try:
                return await func(*args, **kwargs)
            except Exception as e:
                last_error = e
                print(f"{func.__name__} failed: {e}, trying next...")
                continue
        
        raise last_error
```

```python
{self.tool_descriptions}

Instructions:

Response Format:
```

### 5.6 Pipeline编排

```python
# agent/core/pipeline.py

import agentscope
from agentscope.pipelines import (
    sequentialpipeline,
    parallelpipeline,
    ifpipeline,
    switchpipeline
)
from typing import List, Dict, Any, Callable
import asyncio

class SmartLinkPipeline:
    """
    Pipeline编排器
    
    支持：
    - 顺序执行 (Sequential)
    - 并行执行 (Parallel)
    - 条件分支 (If/Else)
    - 多路选择 (Switch/Case)
    """
    
    def __init__(self):
        self.agents: Dict[str, agentscope.Agent] = {}
    
    def register_agent(self, name: str, agent: agentscope.Agent):
        """注册Agent到Pipeline"""
        self.agents[name] = agent
    
    async def execute_sequential(
        self,
        agent_names: List[str],
        initial_input: str,
        **kwargs
    ) -> Any:
        """
        顺序执行Pipeline
        
        前一个Agent的输出作为后一个Agent的输入
        """
        agents = [self.agents[name] for name in agent_names]
        
        # 使用AgentScope的sequentialpipeline
        result = await sequentialpipeline(
            agents,
            initial_input,
            **kwargs
        )
        
        return result
    
    async def execute_parallel(
        self,
        agent_names: List[str],
        input_data: str,
        aggregator: Callable = None
    ) -> Dict[str, Any]:
        """
        并行执行Pipeline
        
        多个Agent同时处理，结果聚合
        """
        agents = [self.agents[name] for name in agent_names]
        
        # 并行执行
        results = await parallelpipeline(
            agents,
            input_data
        )
        
        # 自定义聚合
        if aggregator:
            return aggregator(results)
        
        return results
    
    async def execute_conditional(
        self,
        condition: Callable[[str], bool],
        true_agent: str,
        false_agent: str,
        input_data: str
    ) -> Any:
        """
        条件执行Pipeline
        """
        if condition(input_data):
            agent = self.agents[true_agent]
        else:
            agent = self.agents[false_agent]
        
        return await agent.reply(input_data)
    
    async def execute_switch(
        self,
        selector: Callable[[str], str],
        cases: Dict[str, str],
        default: str,
        input_data: str
    ) -> Any:
        """
        Switch/Case Pipeline
        
        根据选择器结果路由到不同Agent
        """
        case = selector(input_data)
        agent_name = cases.get(case, default)
        agent = self.agents[agent_name]
        
        return await agent.reply(input_data)
```

```python
class IntentRouter:
    """意图路由器"""
    
    def __init__(self):
        self.intents: Dict[str, IntentDefinition] = {}
        self._load_builtin_intents()
    
    def _load_builtin_intents(self):
        """加载内置意图"""
        for intent in BUILTIN_INTENTS:
            self.intents[intent.id] = intent
    
    def register_intent(self, intent: IntentDefinition):
        """注册自定义意图"""
        self.intents[intent.id] = intent
    
    def match_intent(
        self, 
        user_input: str,
        min_confidence: float = 0.6
    ) -> Optional[IntentDefinition]:
        """
        匹配用户输入到意图
        
        使用关键词+模式匹配
        """
        best_match = None
        best_score = 0.0
        
        for intent in self.intents.values():
            if not intent.is_active:
                continue
            
            score = self._calculate_match_score(user_input, intent)
            
            if score > best_score and score >= min_confidence:
                best_score = score
                best_match = intent
        
        return best_match
    
    def _calculate_match_score(
        self, 
        user_input: str, 
        intent: IntentDefinition
    ) -> float:
        """计算匹配分数"""
        score = 0.0
        input_lower = user_input.lower()
        
        # 关键词匹配
        keyword_matches = sum(
            1 for kw in intent.keywords if kw in input_lower
        )
        score += keyword_matches * 0.2
        
        # 正则匹配
        import re
        pattern_matches = sum(
            1 for p in intent.patterns if re.match(p, user_input)
        )
        score += pattern_matches * 0.5
        
        # 示例匹配（使用简单相似度）
        example_scores = [
            self._similarity(input_lower, ex.lower())
            for ex in intent.examples
        ]
        if example_scores:
            score += max(example_scores) * 0.3
        
        return min(score, 1.0)
    
    def _similarity(self, s1: str, s2: str) -> float:
        """计算字符串相似度（简化版）"""
        # 实际应使用更复杂的算法如余弦相似度
        words1 = set(s1.split())
        words2 = set(s2.split())
        intersection = words1 & words2
        union = words1 | words2
        return len(intersection) / len(union) if union else 0.0
```

#### 7.3.1 连接建立

```
连接URL: wss://api.smartlink.com/ws/chat/{client_id}?token={jwt_token}

  - Authorization: Bearer {token}
```

#### 7.3.2 消息协议

| 消息类型 | 方向 | 描述 | 示例 |
|---------|------|------|------|
| `init` | C->S | 初始化会话 | `{"type":"init","data":{"app_id":"app_123"}}` |
| `init_ack` | S->C | 初始化确认 | `{"type":"init_ack","data":{"session_id":"sess_456"}}` |
| `chat` | C->S | 发送聊天消息 | `{"type":"chat","data":{"message":"你好"}}` |
| `token` | S->C | 流式Token | `{"type":"token","data":{"content":"你"}}` |
| `tool_call` | S->C | 工具调用 | `{"type":"tool_call","data":{"tool":"search","input":{}}}` |
| `tool_result` | S->C | 工具结果 | `{"type":"tool_result","data":{"tool":"search","output":{}}}` |
| `complete` | S->C | 完成 | `{"type":"complete","data":{"message_id":"msg_789"}}` |
| `error` | S->C | 错误 | `{"type":"error","data":{"code":5001,"message":"..."}}` |
| `ping` | S->C | 心跳 | `{"type":"ping","data":{"timestamp":1234567890}}` |
| `pong` | C->S | 心跳响应 | `{"type":"pong","data":{"timestamp":1234567890}}` |

```python
# agent/core/memory_manager.py

from typing import List, Dict, Any, Optional
from dataclasses import dataclass, field
from datetime import datetime, timedelta
from enum import Enum
import json
import hashlib

class MemoryType(Enum):
    WORKING = "working"           # 工作记忆
    SHORT_TERM = "short_term"     # 短期记忆
    LONG_TERM = "long_term"       # 长期记忆
    EPISODIC = "episodic"         # 情节记忆
    SEMANTIC = "semantic"         # 语义记忆

@dataclass
class MemoryEntry:
    """记忆条目"""
    id: str
    content: str
    memory_type: MemoryType
    
    # 元数据
    agent_id: Optional[str] = None
    conversation_id: Optional[str] = None
    user_id: Optional[str] = None
    
    # 内容相关
    summary: Optional[str] = None
    importance: int = 50  # 0-100
    
    # 时间戳
    created_at: datetime = field(default_factory=datetime.utcnow)
    expires_at: Optional[datetime] = None
    last_accessed_at: Optional[datetime] = None
    
    # 访问统计
    access_count: int = 0
    
    # 关联
    related_memories: List[str] = field(default_factory=list)
    source_message_id: Optional[str] = None
    
    # 向量嵌入（预留）
    embedding: Optional[List[float]] = None
    
    def to_dict(self) -> Dict[str, Any]:
        """转换为字典"""
        return {
            "id": self.id,
            "content": self.content,
            "memory_type": self.memory_type.value,
            "agent_id": self.agent_id,
            "conversation_id": self.conversation_id,
            "summary": self.summary,
            "importance": self.importance,
            "created_at": self.created_at.isoformat(),
            "expires_at": self.expires_at.isoformat() if self.expires_at else None,
            "access_count": self.access_count
        }


class MemoryManager:
    """
    记忆管理器
    
    统一管理多层记忆系统
    """
    
    def __init__(self, db_session=None):
        self.db = db_session
        
        # 配置
        self.config = {
            "short_term_limit": 20,      # 短期记忆轮数限制
            "working_memory_size": 5,     # 工作记忆条目数
            "importance_threshold": 70,   # 长期记忆提取阈值
            "default_ttl_hours": 24,      # 默认过期时间
            "max_context_tokens": 4000,   # 最大上下文Token数
        }
        
        # 内存缓存
        self._working_memory: Dict[str, List[MemoryEntry]] = {}
        self._cache: Dict[str, MemoryEntry] = {}
    
    async def add_memory(
        self,
        content: str,
        memory_type: MemoryType,
        agent_id: Optional[str] = None,
        conversation_id: Optional[str] = None,
        importance: int = 50,
        ttl_hours: Optional[int] = None,
        metadata: Optional[Dict] = None
    ) -> MemoryEntry:
        """
        添加记忆
        
        Args:
            content: 记忆内容
            memory_type: 记忆类型
            agent_id: Agent ID
            conversation_id: 会话ID
            importance: 重要性评分
            ttl_hours: 过期时间（小时）
            metadata: 额外元数据
        """
        # 生成ID
        memory_id = self._generate_id(content)
        
        # 计算过期时间
        expires_at = None
        if ttl_hours or memory_type == MemoryType.SHORT_TERM:
            hours = ttl_hours or self.config["default_ttl_hours"]
            expires_at = datetime.utcnow() + timedelta(hours=hours)
        
        entry = MemoryEntry(
            id=memory_id,
            content=content,
            memory_type=memory_type,
            agent_id=agent_id,
            conversation_id=conversation_id,
            importance=importance,
            expires_at=expires_at,
            metadata=metadata or {}
        )
        
        # 添加到对应存储
        if memory_type == MemoryType.WORKING:
            await self._add_to_working_memory(entry)
        else:
            # 持久化到数据库
            await self._persist_memory(entry)
        
        # 检查是否需要提升到长期记忆
        if importance >= self.config["importance_threshold"]:
            await self._promote_to_long_term(entry)
        
        return entry
    
    async def get_context(
        self,
        conversation_id: str,
        current_input: str,
        agent_id: Optional[str] = None,
        max_tokens: int = 4000
    ) -> str:
        """
        获取上下文
        
        智能组合多层记忆构建上下文
        """
        context_parts = []
        total_tokens = 0
        
        # 1. 工作记忆（最高优先级）
        working = await self._get_working_memory(conversation_id)
        for entry in working:
            tokens = self._estimate_tokens(entry.content)
            if total_tokens + tokens > max_tokens * 0.3:  # 30%给工作记忆
                break
            context_parts.append(entry.content)
            total_tokens += tokens
            await self._update_access(entry)
        
        # 2. 短期记忆（当前会话历史）
        remaining = max_tokens - total_tokens
        short_term = await self._get_short_term_memory(
            conversation_id, 
            limit=self.config["short_term_limit"]
        )
        for entry in reversed(short_term):  # 从新到旧
            tokens = self._estimate_tokens(entry.content)
            if total_tokens + tokens > max_tokens * 0.8:  # 50%给短期记忆
                break
            context_parts.append(entry.content)
            total_tokens += tokens
            await self._update_access(entry)
        
        # 3. 长期记忆（语义检索）
        if total_tokens < max_tokens:
            remaining = max_tokens - total_tokens
            long_term = await self._retrieve_long_term(
                conversation_id,
                current_input,
                limit=3
            )
            for entry in long_term:
                tokens = self._estimate_tokens(entry.content)
                if total_tokens + tokens > max_tokens:
                    break
                context_parts.append(entry.content)
                total_tokens += tokens
                await self._update_access(entry)
        
        return "\n\n".join(context_parts)
    
    async def summarize_conversation(
        self,
        conversation_id: str
    ) -> Optional[MemoryEntry]:
        """
        总结会话并保存为长期记忆
        """
        # 获取会话消息
        messages = await self._get_conversation_messages(conversation_id)
        
        if len(messages) < 5:
            return None
        
        # 生成摘要（使用LLM）
        summary_prompt = f"""总结以下对话的关键信息：

{self._format_messages(messages)}

请提取：
1. 用户的主要意图
2. 重要的决策和结论
3. 用户偏好

用简洁的JSON格式输出：
{{
    "key_points": ["..."],
    "user_preferences": {{}},
    "decisions": ["..."]
}}"""
        
        # TODO: 调用LLM生成摘要
        summary_content = "Session summary placeholder"
        
        # 保存为长期记忆
        entry = await self.add_memory(
            content=summary_content,
            memory_type=MemoryType.LONG_TERM,
            conversation_id=conversation_id,
            importance=80,
            ttl_hours=None  # 永久存储
        )
        
        return entry
    
    async def clear_conversation_memory(self, conversation_id: str):
        """清除会话相关记忆"""
        # 清除工作记忆
        if conversation_id in self._working_memory:
            del self._working_memory[conversation_id]
        
        # 清除数据库中的短期记忆
        if self.db:
            await self.db.execute(
                "DELETE FROM memories WHERE conversation_id = ? AND memory_type = ?",
                (conversation_id, MemoryType.SHORT_TERM.value)
            )
    
    # ============ 私有方法 ============
    
    async def _add_to_working_memory(self, entry: MemoryEntry):
        """添加到工作记忆"""
        conv_id = entry.conversation_id or "global"
        if conv_id not in self._working_memory:
            self._working_memory[conv_id] = []
        
        self._working_memory[conv_id].append(entry)
        
        # 保持大小限制
        if len(self._working_memory[conv_id]) > self.config["working_memory_size"]:
            self._working_memory[conv_id].pop(0)
    
    async def _persist_memory(self, entry: MemoryEntry):
        """持久化到数据库"""
        if not self.db:
            return
        
        # 插入数据库
        await self.db.execute(
            """INSERT INTO memories 
            (id, agent_id, conversation_id, memory_type, content, summary, importance, 
             expires_at, related_memories, source_message_id, created_at)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)""",
            (
                entry.id, entry.agent_id, entry.conversation_id,
                entry.memory_type.value, entry.content, entry.summary,
                entry.importance, entry.expires_at,
                json.dumps(entry.related_memories),
                entry.source_message_id, entry.created_at
            )
        )
    
    async def _get_working_memory(
        self, 
        conversation_id: str
    ) -> List[MemoryEntry]:
        """获取工作记忆"""
        return self._working_memory.get(conversation_id, [])
    
    async def _get_short_term_memory(
        self,
        conversation_id: str,
        limit: int = 20
    ) -> List[MemoryEntry]:
        """获取短期记忆"""
        if not self.db:
            return []
        
        rows = await self.db.fetchall(
            """SELECT * FROM memories 
            WHERE conversation_id = ? AND memory_type = ?
            ORDER BY created_at DESC LIMIT ?""",
            (conversation_id, MemoryType.SHORT_TERM.value, limit)
        )
        
        return [self._row_to_entry(row) for row in rows]
    
    async def _retrieve_long_term(
        self,
        conversation_id: str,
        query: str,
        limit: int = 3
    ) -> List[MemoryEntry]:
        """检索长期记忆"""
        if not self.db:
            return []
        
        # 关键词匹配
        keywords = query.split()
        placeholders = ','.join(['?'] * len(keywords))
        
        rows = await self.db.fetchall(
            f"""SELECT * FROM memories 
            WHERE memory_type = ? AND content LIKE '%' || ? || '%'
            ORDER BY importance DESC, created_at DESC LIMIT ?""",
            (MemoryType.LONG_TERM.value, keywords[0], limit)
        )
        
        return [self._row_to_entry(row) for row in rows]
    
    async def _promote_to_long_term(self, entry: MemoryEntry):
        """将记忆提升为长期记忆"""
        if entry.memory_type == MemoryType.LONG_TERM:
            return
        
        # 生成摘要
        summary = entry.summary or entry.content[:200]
        
        # 创建新的长期记忆条目
        await self.add_memory(
            content=summary,
            memory_type=MemoryType.LONG_TERM,
            agent_id=entry.agent_id,
            conversation_id=entry.conversation_id,
            importance=entry.importance,
            ttl_hours=None
        )
    
    async def _update_access(self, entry: MemoryEntry):
        """更新访问统计"""
        entry.access_count += 1
        entry.last_accessed_at = datetime.utcnow()
    
    def _generate_id(self, content: str) -> str:
        """生成记忆ID"""
        return hashlib.md5(content.encode()).hexdigest()[:16]
    
    def _estimate_tokens(self, text: str) -> int:
        """估算Token数"""
        return len(text) // 4
    
    def _row_to_entry(self, row: Dict) -> MemoryEntry:
        """数据库行转换为MemoryEntry"""
        return MemoryEntry(
            id=row["id"],
            content=row["content"],
            memory_type=MemoryType(row["memory_type"]),
            agent_id=row.get("agent_id"),
            conversation_id=row.get("conversation_id"),
            summary=row.get("summary"),
            importance=row.get("importance", 50),
            created_at=row["created_at"],
            expires_at=row.get("expires_at"),
            access_count=row.get("access_count", 0),
            related_memories=json.loads(row.get("related_memories", "[]"))
        )
    
    def _format_messages(self, messages: List[Dict]) -> str:
        """格式化消息列表"""
        return "\n".join([
            f"{m['role']}: {m['content'][:100]}"
            for m in messages
        ])
```

### 8.4 记忆上下文窗口管理

```python
# agent/core/context_window.py

from typing import List, Dict, Any
from dataclasses import dataclass

@dataclass
class ContextWindow:
    """
    上下文窗口管理
    
    管理LLM的上下文Token限制
    """
    
    max_tokens: int = 4000
    reserve_tokens: int = 500  # 预留token给响应
    
    def build_prompt(
        self,
        system_prompt: str,
        memories: List[str],
        current_message: str,
        tools_description: str = ""
    ) -> str:
        """
        构建符合Token限制的提示词
        
        策略：
        1. 系统提示词（必须保留）
        2. 工具描述（如需要）
        3. 当前消息（必须保留）
        4. 历史记忆（按重要性排序，裁剪）
        """
        available = self.max_tokens - self.reserve_tokens
        
        # 必须保留的内容
        fixed_parts = [
            system_prompt,
            tools_description,
            current_message
        ]
        
        fixed_tokens = sum(
            self._count_tokens(p) for p in fixed_parts if p
        )
        
        available_for_memories = available - fixed_tokens
        
        # 选择记忆
        selected_memories = []
        memory_tokens = 0
        
        for memory in memories:
            tokens = self._count_tokens(memory)
            if memory_tokens + tokens <= available_for_memories:
                selected_memories.append(memory)
                memory_tokens += tokens
            else:
                break
        
        # 构建最终提示词
        parts = [system_prompt]
        
        if tools_description:
            parts.append(f"\n\nAvailable Tools:\n{tools_description}")
        
        if selected_memories:
            parts.append(f"\n\nPrevious Context:\n" + "\n".join(selected_memories))
        
        parts.append(f"\n\nCurrent Input:\n{current_message}")
        
        return "\n".join(parts)
    
    def _count_tokens(self, text: str) -> int:
        """统计Token数（粗略估计）"""
        return len(text) // 4
```

```python
class StructuredFormatter(logging.Formatter):
    """结构化JSON日志格式化器"""
    
    def format(self, record: logging.LogRecord) -> str:
        log_data = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": record.levelname,
            "message": record.getMessage(),
            "logger": record.name,
            "module": record.module,

            "tenant_id": tenant_id_var.get(),

        }
        
        # 异常信息
        if record.exc_info:
            log_data["exception"] = {
                "type": record.exc_info[0].__name__,
                "message": str(record.exc_info[1]),
                "stacktrace": self.formatException(record.exc_info)
            }

        self.logger.warning(message, extra=extra)
    
    def debug(self, message: str, **kwargs):
```

```python
# agent/llm/cache.py

from typing import Dict, Any, Optional
import hashlib
import json
from datetime import timedelta
import redis

class LLMResponseCache:
    """LLM响应缓存"""
    
    def __init__(self, redis_client: redis.Redis):
        self.redis = redis_client
        self.default_ttl = timedelta(hours=24)
    
    def _generate_cache_key(
        self,
        prompt: str,
        model: str,
        temperature: float = 0.7
    ) -> str:
        """生成缓存键"""
        cache_data = {
            "prompt": prompt,
            "model": model,
            "temperature": temperature
        }
        hash_key = hashlib.sha256(
            json.dumps(cache_data, sort_keys=True).encode()
        ).hexdigest()
        return f"llm_cache:{model}:{hash_key}"
    
    async def get(
        self,
        prompt: str,
        model: str,
        temperature: float = 0.7
    ) -> Optional[str]:
        """获取缓存"""
        key = self._generate_cache_key(prompt, model, temperature)
        cached = self.redis.get(key)
        if cached:
            return json.loads(cached)
        return None
    
    async def set(
        self,
        prompt: str,
        model: str,
        response: str,
        temperature: float = 0.7,
        ttl: timedelta = None
    ):
        """设置缓存"""
        key = self._generate_cache_key(prompt, model, temperature)
        self.redis.setex(
            key,
            ttl or self.default_ttl,
            json.dumps(response)
        )
    
    async def clear_for_model(self, model: str):
        """清除特定模型的缓存"""
        pattern = f"llm_cache:{model}:*"
        keys = self.redis.keys(pattern)
        if keys:
            self.redis.delete(*keys)
```

```python
# gateway/middleware/rate_limiter.py

from fastapi import Request, HTTPException
from typing import Dict
import redis
import time

class RateLimiter:
    """分布式限流器"""
    
    def __init__(self, redis_client: redis.Redis):
        self.redis = redis_client
        
        # 限流规则
        self.rules = {
            "default": {"requests": 100, "window": 60},      # 100次/分钟
            "chat": {"requests": 20, "window": 60},          # 20次/分钟
            "stream": {"requests": 10, "window": 60},        # 10次/分钟
            "api_key": {"requests": 500, "window": 3600},    # 500次/小时
        }
    
    async def check_rate_limit(
        self,
        request: Request,
        tenant_id: str,
        endpoint_type: str = "default"
    ) -> bool:
        """检查限流"""
        rule = self.rules.get(endpoint_type, self.rules["default"])
        
        # 生成限流键
        key = f"rate_limit:{tenant_id}:{endpoint_type}"
        
        # 滑动窗口算法
        now = time.time()
        window_start = now - rule["window"]
        
        # 移除过期记录
        self.redis.zremrangebyscore(key, 0, window_start)
        
        # 获取当前窗口内请求数
        current_count = self.redis.zcard(key)
        
        if current_count >= rule["requests"]:
            raise HTTPException(
                status_code=429,
                detail=f"Rate limit exceeded. Retry after {rule['window']} seconds"
            )
        
        # 记录本次请求
        self.redis.zadd(key, {str(now): now})
        self.redis.expire(key, rule["window"])
        
        return True
```

## 12. 附录 - 架构决策记录

### 12.1 ADR (Architecture Decision Records)

#### ADR-001: 选择三层架构

| 项目 | 内容 |
|------|------|
| **状态** | 已采纳 |
| **背景** | 需要支持多Agent协作、LLM调用、实时通信 |
| **决策** | 采用Gateway → Service → Agent三层架构 |
| **理由** | 1. 关注点分离，职责清晰<br>2. 便于独立演进和测试<br>3. 支持水平扩展 |
| **后果** | + 清晰的边界定义<br>+ 易于维护<br>- 需要更多的抽象层 |

#### ADR-002: 选择LiteLLM作为LLM代理

| 项目 | 内容 |
|------|------|
| **状态** | 已采纳 |
| **背景** | 需要支持OpenAI、Claude、Ollama等多种LLM |
| **决策** | 使用LiteLLM作为统一的LLM调用层 |
| **理由** | 1. 统一API，减少适配成本<br>2. 自动重试和降级<br>3. 支持本地模型 |
| **后果** | + 减少LLM切换成本<br>+ 内置容错机制<br>- 增加一层依赖 |

#### ADR-003: 选择SQLite作为MVP存储

| 项目 | 内容 |
|------|------|
| **状态** | 已采纳 (过渡方案) |
| **背景** | MVP阶段需要快速验证，降低部署复杂度 |
| **决策** | Phase1使用SQLite，Phase2迁移PostgreSQL |
| **理由** | 1. 无额外依赖，快速启动<br>2. 适合单节点验证<br>3. 可平滑迁移 |
| **后果** | + 快速迭代<br>- 不支持高并发<br>- 需要迁移计划 |

#### ADR-004: 选择AgentScope作为Agent框架补充

| 项目 | 内容 |
|------|------|
| **状态** | 计划采纳 |
| **背景** | 需要更强大的多Agent协作能力 |
| **决策** | 集成AgentScope的MessageHub和Pipeline能力 |
| **理由** | 1. 阿里开源，本土化支持好<br>2. Pipeline编排能力强<br>3. ReActAgent成熟 |
| **后果** | + 增强协作能力<br>+ 减少自研成本<br>- 需要适配层 |

#### ADR-005: 选择FastAPI作为Web框架

| 项目 | 内容 |
|------|------|
| **状态** | 已采纳 |
| **背景** | 要高性能异步框架，支持WebSocket |
| **决策** | 使用FastAPI作为Gateway层框架 |
| **理由** | 1. 原生异步支持<br>2. 自动API文档<br>3. 类型安全 |
| **后果** | + 高性能<br>+ 自动文档<br>- 团队需学习异步模式 |

### 12.2 技术选型对照表
