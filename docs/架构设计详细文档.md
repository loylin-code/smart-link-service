## 1. 概述

### 1.1 项目背景

SmartLink是一个正在从单租户原型演进为企业级多租户Agent应用管理平台的后端服务。平台采用类似OpenClaw的架构模式，通过集中式网关管理、分布式Agent执行，为企业提供可扩展的AI Agent应用解决方案。

**核心定位**：SmartLink不仅仅是一个"对话式AI"，而是一个能够**真正执行任务**的Agent平台。用户可以通过自然语言描述需求，Agent理解意图后自动规划并执行任务，最终通过前端渲染引擎展示结果。

### 1.2 与OpenClaw的架构对比

| 维度 | OpenClaw | SmartLink | 说明 |
|------|----------|-----------|------|
| **定位** | 个人AI助手网关 | 企业Agent应用平台 | SmartLink更侧重企业级场景 |
| **渠道** | Telegram/WhatsApp/Discord | Web前端/企业IM/开放API | SmartLink支持更多企业渠道 |
| **租户** | 单用户 | 多租户 | SmartLink支持租户隔离 |
| **前端** | 纯文本响应 | LLM生成前端组件 | SmartLink支持富交互UI |
| **部署** | 本地优先 | 云端部署 | SmartLink支持水平扩展 |

### 1.3 设计目标

| 目标 | 描述 | 指标 |
|------|------|------|
| **企业级就绪** | 多租户架构、完整的安全体系、企业级集成 | 支持10+租户 |
| **高可扩展** | 水平扩展能力、分布式部署 | 1,000+并发WebSocket |
| **灵活编排** | 插件化技能系统、可定制工作流 | 支持自定义Skill |
| **生产可靠** | 高可用、容错、监控告警 | 99.9%可用性目标 |
| **前端智能** | LLM生成前端组件、动态渲染 | 声明式组件系统 |

### 1.3 核心架构决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 租户隔离策略 | 共享数据库 + tenant_id字段 | 简单易迁移，适合10租户规模 |
| 认证模型 | OAuth2混合模式 + JWT + API Keys | 支持企业SSO、第三方登录、服务间通信 |
| 会话管理 | Redis Cluster | 支持水平扩展、Pub/Sub消息分发 |
| Agent分发 | 任务队列 + Agent池 | 负载均衡、配额管理、自动扩缩容 |
| 前端响应 | 声明式组件JSON + 通用渲染器 | 灵活性高、框架无关 |
| 数据库 | PostgreSQL + 行级安全(RLS) | ACID事务、成熟生态、多租户支持 |

### 1.4 关键技术参考

本架构设计参考了以下项目和模式：

- **OpenClaw**: 网关中心化设计、Lane并发模型、WebSocket协议
- **A2A Protocol**: Agent间通信协议标准
- **LangGraph**: Agent工作流编排模式
- **Vercel AI SDK**: 流式响应和结构化输出

### 3.2 标准消息格式

```python
# schemas/channel.py
from pydantic import BaseModel
from typing import Optional, List, Dict, Any
from datetime import datetime
from enum import Enum

class ChannelType(str, Enum):
    """渠道类型"""
    WEB = "web"              # Web前端WebSocket
    DINGTALK = "dingtalk"    # 钉钉机器人
    FEISHU = "feishu"        # 飞书机器人
    WECOM = "wecom"          # 企业微信
    API = "api"              # 开放API
    MOBILE = "mobile"        # 移动端

class MessageType(str, Enum):
    """消息类型"""
    TEXT = "text"            # 文本消息
    IMAGE = "image"          # 图片消息
    FILE = "file"            # 文件消息
    CARD = "card"            # 卡片消息
    EVENT = "event"          # 事件消息

class Attachment(BaseModel):
    """附件信息"""
    type: str                # image, file, audio, video
    url: Optional[str]       # 附件URL
    content: Optional[bytes] # 附件内容(小文件)
    name: Optional[str]      # 文件名
    size: Optional[int]      # 文件大小

class StandardMessage(BaseModel):
    """标准化消息格式 - 所有渠道转换为统一格式"""
    
    # 渠道信息
    channel_id: str                    # 渠道标识 (web, dingtalk_xxx, api_xxx)
    channel_type: ChannelType          # 渠道类型
    
    # 用户信息
    user_id: str                       # 用户ID (渠道内唯一)
    tenant_id: str                     # 租户ID
    
    # 会话信息
    conversation_id: str               # 对话ID
    session_key: Optional[str]         # WebSocket会话Key
    
    # 消息内容
    message_type: MessageType = MessageType.TEXT
    content: str                       # 消息文本内容
    attachments: List[Attachment] = [] # 附件列表
    
    # 元数据
    timestamp: datetime                # 消息时间戳
    metadata: Dict[str, Any] = {}      # 渠道特定元数据
    
    # 原始数据(用于调试)
    raw_data: Optional[Dict[str, Any]] = None

class ChannelAdapter(ABC):
    """Channel适配器基类"""
    
    channel_type: ChannelType
    
    @abstractmethod
    async def parse_message(self, raw_data: Dict[str, Any]) -> StandardMessage:
        """解析原始消息为标准格式"""
        pass
    
    @abstractmethod
    async def send_message(self, message: StandardMessage) -> bool:
        """发送消息到渠道"""
        pass
    
    @abstractmethod
    async def validate_signature(self, request: Request) -> bool:
        """验证请求签名"""
        pass
    
    async def handle_webhook(self, request: Request) -> Response:
        """处理Webhook回调"""
        pass
```

### 3.3 WebChannel实现

```python
# gateway/channels/web.py
from gateway.channels.base import ChannelAdapter, StandardMessage, ChannelType
from fastapi import WebSocket, WebSocketDisconnect
from typing import Dict, Any
import json

class WebChannelAdapter(ChannelAdapter):
    """Web前端WebSocket渠道适配器"""
    
    channel_type = ChannelType.WEB
    
    def __init__(self, tenant_id: str):
        self.tenant_id = tenant_id
        self.connections: Dict[str, WebSocket] = {}  # session_key -> WebSocket
    
    async def connect(self, websocket: WebSocket, session_key: str):
        """建立WebSocket连接"""
        await websocket.accept()
        self.connections[session_key] = websocket
    
    async def disconnect(self, session_key: str):
        """断开连接"""
        if session_key in self.connections:
            del self.connections[session_key]
    
    async def parse_message(self, raw_data: Dict[str, Any]) -> StandardMessage:
        """解析WebSocket消息"""
        return StandardMessage(
            channel_id=f"web:{self.tenant_id}",
            channel_type=ChannelType.WEB,
            user_id=raw_data.get("user_id", ""),
            tenant_id=self.tenant_id,
            conversation_id=raw_data.get("conversation_id", ""),
            session_key=raw_data.get("session_key"),
            message_type=raw_data.get("type", "text"),
            content=raw_data.get("content", ""),
            timestamp=datetime.utcnow(),
            metadata={
                "client_info": raw_data.get("client_info", {}),
                "app_id": raw_data.get("app_id")
            },
            raw_data=raw_data
        )
    
    async def send_message(self, message: StandardMessage) -> bool:
        """发送消息到WebSocket"""
        session_key = message.session_key
        if session_key not in self.connections:
            return False
        
        websocket = self.connections[session_key]
        try:
            await websocket.send_json({
                "type": "response",
                "data": {
                    "content": message.content,
                    "attachments": [a.dict() for a in message.attachments]
                },
                "timestamp": message.timestamp.isoformat()
            })
            return True
        except Exception:
            await self.disconnect(session_key)
            return False
    
    async def send_stream(self, session_key: str, chunk: str, done: bool = False):
        """发送流式响应"""
        if session_key not in self.connections:
            return
        
        websocket = self.connections[session_key]
        await websocket.send_json({
            "type": "stream",
            "data": {
                "delta": chunk,
                "done": done
            }
        })
    
    async def validate_signature(self, request: Request) -> bool:
        """WebSocket不需要签名验证，使用JWT"""
        return True
```

```python
# gateway/channels/dingtalk.py
import hashlib
import hmac
import time
from gateway.channels.base import ChannelAdapter, StandardMessage, ChannelType

class DingTalkChannelAdapter(ChannelAdapter):
    """钉钉机器人渠道适配器"""
    
    channel_type = ChannelType.DINGTALK
    
    def __init__(self, app_key: str, app_secret: str, tenant_id: str):
        self.app_key = app_key
        self.app_secret = app_secret
        self.tenant_id = tenant_id
    
    async def parse_message(self, raw_data: Dict[str, Any]) -> StandardMessage:
        """解析钉钉消息"""
        # 钉钉消息格式
        # {
        #   "msgtype": "text",
        #   "text": {"content": "消息内容"},
        #   "msgId": "xxx",
        #   "createAt": 1234567890,
        #   "conversationId": "xxx",
        #   "senderId": "xxx",
        #   "senderNick": "用户昵称"
        # }
        
        msgtype = raw_data.get("msgtype", "text")
        content = ""
        attachments = []
        
        if msgtype == "text":
            content = raw_data.get("text", {}).get("content", "")
        elif msgtype == "picture":
            attachments.append(Attachment(
                type="image",
                url=raw_data.get("content", {}).get("downloadCode", "")
            ))
        
        return StandardMessage(
            channel_id=f"dingtalk:{self.app_key}",
            channel_type=ChannelType.DINGTALK,
            user_id=raw_data.get("senderId", ""),
            tenant_id=self.tenant_id,
            conversation_id=raw_data.get("conversationId", ""),
            message_type=msgtype,
            content=content,
            attachments=attachments,
            timestamp=datetime.fromtimestamp(raw_data.get("createAt", 0) / 1000),
            metadata={
                "msg_id": raw_data.get("msgId"),
                "sender_nick": raw_data.get("senderNick"),
                "is_in_at_list": raw_data.get("isInAtList", False)
            },
            raw_data=raw_data
        )
    
    async def send_message(self, message: StandardMessage) -> bool:
        """发送钉钉消息"""
        import httpx
        
        # 获取access_token
        token = await self._get_access_token()
        
        # 发送消息
        url = f"https://oapi.dingtalk.com/robot/send?access_token={token}"
        
        payload = {
            "msgtype": "text",
            "text": {
                "content": message.content
            },
            "at": {
                "atUserIds": [message.user_id]
            }
        }
        
        async with httpx.AsyncClient() as client:
            response = await client.post(url, json=payload)
            return response.status_code == 200
    
    async def validate_signature(self, request: Request) -> bool:
        """验证钉钉签名"""
        timestamp = request.headers.get("timestamp", "")
        sign = request.headers.get("sign", "")
        
        if not timestamp or not sign:
            return False
        
        # 防重放攻击：时间戳误差不超过5分钟
        if abs(time.time() - int(timestamp) / 1000) > 300:
            return False
        
        # 计算签名
        string_to_sign = f"{timestamp}\n{self.app_secret}"
        hmac_code = hmac.new(
            self.app_secret.encode(),
            string_to_sign.encode(),
            digestmod=hashlib.sha256
        ).digest()
        
        expected_sign = base64.b64encode(hmac_code).decode()
        return sign == expected_sign
    
    async def handle_webhook(self, request: Request) -> Response:
        """处理钉钉Webhook回调"""
        # 验证签名
        if not await self.validate_signature(request):
            return Response(status_code=401, content="Invalid signature")
        
        # 解析消息
        raw_data = await request.json()
        
        # 处理文本消息
        if raw_data.get("msgtype") == "text":
            message = await self.parse_message(raw_data)
            # 路由到Gateway处理
            await self._route_to_gateway(message)
        
        return Response(status_code=200)
    
    async def _get_access_token(self) -> str:
        """获取钉钉access_token"""
        # 实现获取token的逻辑
        pass

    def get_adapter(cls, channel_id: str) -> Optional[ChannelAdapter]:

        channel_type: ChannelType, 
        config: Dict[str, Any]

        factory = cls._factories.get(channel_type)
        if not factory:

        cls._adapters[adapter.channel_id] = adapter

网关层作为系统的统一入口，负责WebSocket连接管理、HTTP REST API路由、认证授权等核心职责。采用FastAPI作为基础框架，结合Redis Cluster实现分布式会话管理。

### 4.1 WebSocket协议设计

#### 4.1.1 消息帧结构

WebSocket通信采用JSON消息帧，所有消息遵循统一的格式规范：

```typescript
// 基础消息帧结构
interface WebSocketMessage {
  type: MessageType;           // 消息类型
  timestamp: number;           // 发送时间戳 (Unix毫秒)
  sequence?: number;           // 序列号，用于消息排序
  sessionKey?: string;         // 会话标识 (连接后分配)
  laneId?: string;             // 并发通道标识
  data: unknown;               // 消息载荷
}

// 消息类型枚举
type MessageType = 
  | 'handshake'       // 连接握手
  | 'ack'             // 握手确认
  | 'chat'            // 用户聊天消息
  | 'response'        // AI响应
  | 'component'       // 前端组件
  | 'heartbeat'       // 心跳
  | 'heartbeat_ack'   // 心跳确认
  | 'error'           // 错误消息
  | 'status';         // 状态更新
```

**消息类型详细说明：**

| 类型 | 方向 | 描述 | 关键字段 |
|------|------|------|----------|
| `handshake` | C→S | 连接握手，携带认证信息 | `data: {api_key, tenant_id, user_id}` |
| `ack` | S→C | 握手确认，分配sessionKey | `data: {sessionKey, expires_at}` |
| `chat` | C→S | 用户发送聊天消息 | `data: {message, app_id, context}` |
| `response` | S→C | AI流式或完整响应 | `data: {content, done, delta}` |
| `component` | S→C | 前端组件JSON | `data: {type, props, children}` |
| `heartbeat` | C→S | 客户端心跳 | 无data字段或空对象 |
| `heartbeat_ack` | S→C | 服务端心跳响应 | `data: {server_time}` |
| `error` | S→C | 错误通知 | `data: {code, message, details}` |
| `status` | S→C | 状态更新 | `data: {stage, progress, metadata}` |

**服务端心跳检测实现：**

```python
# gateway/websocket/heartbeat.py
import asyncio
from typing import Dict
from datetime import datetime, timedelta

class HeartbeatManager:
    """WebSocket连接心跳管理器"""
    
    def __init__(self, timeout_seconds: int = 60):
        self.connections: Dict[str, dict] = {}
        self.timeout = timeout_seconds
        self._monitor_task = None
    
    async def start_monitoring(self):
        """启动心跳监控循环"""
        self._monitor_task = asyncio.create_task(self._monitor_loop())
    
    async def _monitor_loop(self):
        """定期检查连接活跃度"""
        while True:
            await asyncio.sleep(30)
            now = datetime.utcnow()
            dead_connections = []
            
            for session_key, conn in self.connections.items():
                last_ping = conn['last_ping']
                if now - last_ping > timedelta(seconds=self.timeout):
                    dead_connections.append(session_key)
            
            for session_key in dead_connections:
                await self._close_dead_connection(session_key)
    
    def record_ping(self, session_key: str):
        """记录客户端心跳"""
        if session_key in self.connections:
            self.connections[session_key]['last_ping'] = datetime.utcnow()
    
    async def _close_dead_connection(self, session_key: str):
        """关闭超时连接"""
        if session_key in self.connections:
            ws = self.connections[session_key]['websocket']
            await ws.close(code=1001, reason="Heartbeat timeout")
            del self.connections[session_key]
```

sessionKey采用分层编码，便于路由和调试：

```

- tenant_id: 租户标识 (base62编码)

import base64
import time
from datetime import datetime

class SessionKeyGenerator:
    """会话键生成器"""
    
    @staticmethod
    def generate(tenant_id: str) -> str:
        """生成新的sessionKey"""
        timestamp = int(time.time())
        random_bytes = secrets.token_bytes(6)
        random_str = base64.urlsafe_b64encode(random_bytes).decode().rstrip('=')
        
        # 清理tenant_id中的特殊字符
        safe_tenant = tenant_id.replace(':', '_').replace('/', '_')
        return f"ws:{safe_tenant}:{timestamp}:{random_str}"
    
    @staticmethod
    def parse(session_key: str) -> dict:
        """解析sessionKey"""
        parts = session_key.split(':')
        if len(parts) != 4:
            raise ValueError(f"Invalid session key format: {session_key}")
        
        return {
            "prefix": parts[0],
            "tenant_id": parts[1],
            "timestamp": int(parts[2]),
            "random": parts[3],
            "created_at": datetime.fromtimestamp(int(parts[2]))
        }

# gateway/websocket/pubsub.py
import json
import redis.asyncio as redis
from typing import Callable

class GatewayPubSub:
    """网关间消息广播"""
    
    CHANNEL_PREFIX = "gateway:broadcast"
    
    def __init__(self, redis_client: redis.Redis, gateway_id: str):
        self.redis = redis_client
        self.gateway_id = gateway_id
        self._pubsub = None
        self._handlers: dict[str, Callable] = {}
    
    async def subscribe(self):
        """订阅广播频道"""
        self._pubsub = self.redis.pubsub()
        await self._pubsub.subscribe(f"{self.CHANNEL_PREFIX}:{self.gateway_id}")
        
        # 启动消息监听
        asyncio.create_task(self._listen())
    
    async def _listen(self):
        """监听广播消息"""
        async for message in self._pubsub.listen():
            if message['type'] == 'message':
                data = json.loads(message['data'])
                await self._handle_broadcast(data)
    
    async def broadcast(self, target_session: str, message: dict):
        """广播消息到目标会话所在网关"""
        # 查询sessionKey对应的gateway_id
        session_data = await self.redis.hgetall(f"ws:session:{target_session}")
        if not session_data:
            return
        
        target_gateway = session_data.get(b'gateway_id', b'').decode()
        
        if target_gateway == self.gateway_id:
            # 本地处理
            await self._send_to_local(target_session, message)
        else:
            # 远程广播
            await self.redis.publish(
                f"{self.CHANNEL_PREFIX}:{target_gateway}",
                json.dumps({
                    "session_key": target_session,
                    "message": message,
                    "from_gateway": self.gateway_id
                })
            )
    
    async def _handle_broadcast(self, data: dict):
        """处理接收到的广播消息"""
        session_key = data['session_key']
        message = data['message']
        await self._send_to_local(session_key, message)

# gateway/websocket/lane.py
import asyncio
from enum import Enum
from typing import Optional

class LaneStatus(Enum):
    IDLE = "idle"           # 空闲
    BUSY = "busy"           # 处理中
    QUEUED = "queued"       # 有排队任务

class LaneManager:
    """并发通道管理器"""
    
    MAX_LANES = 3  # 每用户最大并发通道数
    
    def __init__(self, session_key: str, redis_client):
        self.session_key = session_key
        self.redis = redis_client
        self.lanes = {f"lane_{i}": LaneStatus.IDLE for i in range(1, self.MAX_LANES + 1)}
    
    async def acquire_lane(self, priority: int = 0) -> Optional[str]:
        """
        获取可用lane，返回laneId或None
        优先级高的请求优先获取lane
        """
        # 检查是否有空闲lane
        for lane_id, status in self.lanes.items():
            if status == LaneStatus.IDLE:
                self.lanes[lane_id] = LaneStatus.BUSY
                await self._update_lane_status(lane_id, LaneStatus.BUSY)
                return lane_id
        
        # 无可用lane，加入等待队列
        return None
    
    async def release_lane(self, lane_id: str):
        """释放lane"""
        if lane_id in self.lanes:
            self.lanes[lane_id] = LaneStatus.IDLE
            await self._update_lane_status(lane_id, LaneStatus.IDLE)
            
            # 通知等待队列
            await self._notify_waiting_requests(lane_id)
    
    async def _update_lane_status(self, lane_id: str, status: LaneStatus):
        """更新Redis中的lane状态"""
        session_hash_key = f"ws:session:{self.session_key}"
        lanes_data = await self.redis.hget(session_hash_key, "lanes")
        
        if lanes_data:
            import json
            lanes = json.loads(lanes_data)
            lanes[lane_id] = status.value
            await self.redis.hset(session_hash_key, "lanes", json.dumps(lanes))
    
    def get_lane_queue_key(self, lane_id: str) -> str:
        """获取lane对应的任务队列键"""
        return f"queue:{self.session_key}:{lane_id}"

**完整请求路由流程：**

```python
# gateway/websocket/router.py
class RequestRouter:
    """WebSocket请求路由器"""
    
    async def route_request(self, session_key: str, message: dict) -> dict:
        """
        路由请求到适当的Agent
        
        流程:
        1. 验证会话有效性
        2. 获取或分配lane
        3. 创建任务并入队
        4. 返回任务ID和状态
        """
        # 1. 验证会话
        session = await self._validate_session(session_key)
        if not session:
            raise SessionNotFoundError(f"Invalid session: {session_key}")
        
        # 2. 配额检查
        tenant_id = session['tenant_id']
        if not await self._check_quota(tenant_id):
            raise QuotaExceededError("Tenant quota exceeded")
        
        # 3. 获取lane
        lane_manager = LaneManager(session_key, self.redis)
        lane_id = await lane_manager.acquire_lane(
            priority=message.get('priority', 0)
        )
        
        # 4. 创建任务
        task = Task(
            id=self._generate_task_id(),
            session_key=session_key,
            lane_id=lane_id or "queued",
            tenant_id=tenant_id,
            user_id=session['user_id'],
            app_id=message['data'].get('app_id'),
            message=message['data'],
            priority=message.get('priority', 0),
            created_at=datetime.utcnow()
        )
        
        # 5. 入队或立即处理
        if lane_id:
            # 立即分发到Agent
            await self._dispatch_to_agent(task)
        else:
            # 加入等待队列
            await self._enqueue_task(task, lane_manager)
        
        return {
            "task_id": task.id,
            "status": "dispatched" if lane_id else "queued",
            "lane_id": lane_id,
            "estimated_wait": await self._estimate_wait_time(tenant_id)
        }
    
    async def _dispatch_to_agent(self, task: Task):
        """分发任务到Agent池"""
        await self.redis.lpush(
            f"agent:task_queue:{task.tenant_id}",
            task.to_json()
        )
        # 通知Agent调度器
        await self.redis.publish("agent:new_task", task.tenant_id)
```

## 5. Lane并发模型（参考OpenClaw）

> **设计参考**: OpenClaw的Lane-based Command Queue模式，解决并发请求的Race Condition问题。

### 5.1 为什么需要Lane模型？

在多用户、多请求的Agent系统中，并发控制是一个关键问题：

**传统async/await的问题：**
- 并行执行会带来Race Condition
- 多个请求同时读写同一资源导致数据不一致
- 共享状态在并行执行下出现不可预测的行为

**OpenClaw的解决方案：**
> "A simple async setup per agent will leave you with a dump of interleaved garbage." — Cognition (Devin)

OpenClaw采用**预序列化**的设计哲学：默认序列化执行，只有明确安全的任务才允许并行。

**心智模型转变：**
- 传统思路："我要锁什么？"
- Lane模型思路："什么可以并行？"

### 5.2 Lane模型架构

### 5.3 Lane数据结构

```python
# gateway/websocket/lane.py
import asyncio
from enum import Enum
from typing import Optional, Dict, List
from dataclasses import dataclass
from datetime import datetime

class LaneStatus(Enum):
    """Lane状态"""
    IDLE = "idle"           # 空闲，可接受任务
    BUSY = "busy"           # 忙碌，正在处理任务
    QUEUED = "queued"       # 有排队任务

@dataclass
class Lane:
    """Lane定义"""
    id: str                         # Lane ID
    session_key: str                # 所属Session
    status: LaneStatus = LaneStatus.IDLE
    current_task: Optional[str] = None  # 当前任务ID
    queue: List[str] = None         # 排队任务ID列表
    created_at: datetime = None
    
    def __post_init__(self):
        if self.queue is None:
            self.queue = []
        if self.created_at is None:
            self.created_at = datetime.utcnow()

class LaneManager:
    """Lane管理器"""
    
    MAX_LANES = 3  # 每用户最大并发Lane数
    
    def __init__(self, session_key: str, redis_client):
        self.session_key = session_key
        self.redis = redis_client
        self.lanes: Dict[str, Lane] = {}
        self._lock = asyncio.Lock()
        
        # 初始化Lanes
        for i in range(1, self.MAX_LANES + 1):
            lane_id = f"lane_{i}"
            self.lanes[lane_id] = Lane(
                id=lane_id,
                session_key=session_key
            )
    
    async def acquire_lane(self, priority: int = 0) -> Optional[str]:
        """
        获取可用Lane
        
        Args:
            priority: 任务优先级（0最高）
            
        Returns:
            lane_id 或 None（无可用Lane）
        """
        async with self._lock:
            # 优先找空闲Lane
            for lane_id, lane in self.lanes.items():
                if lane.status == LaneStatus.IDLE:
                    lane.status = LaneStatus.BUSY
                    await self._persist_lane_status(lane)
                    return lane_id
            
            # 无空闲Lane，返回None
            return None
    
    async def release_lane(self, lane_id: str, task_id: str):
        """
        释放Lane
        
        Args:
            lane_id: Lane ID
            task_id: 完成的任务ID
        """
        async with self._lock:
            if lane_id not in self.lanes:
                return
            
            lane = self.lanes[lane_id]
            
            # 检查是否有排队任务
            if lane.queue:
                # 取出下一个任务，Lane保持BUSY
                next_task = lane.queue.pop(0)
                lane.current_task = next_task
                await self._notify_task_start(next_task, lane_id)
            else:
                # 无排队任务，Lane变为IDLE
                lane.status = LaneStatus.IDLE
                lane.current_task = None
            
            await self._persist_lane_status(lane)
    
    async def enqueue_task(self, task_id: str, priority: int = 0) -> str:
        """
        将任务加入等待队列
        
        Args:
            task_id: 任务ID
            priority: 优先级
            
        Returns:
            排队的Lane ID
        """
        async with self._lock:
            # 找到队列最短的Lane
            best_lane = min(
                self.lanes.values(),
                key=lambda l: len(l.queue)
            )
            
            best_lane.queue.append(task_id)
            best_lane.status = LaneStatus.QUEUED
            
            await self._persist_lane_status(best_lane)
            return best_lane.id
    
    async def get_lane_status(self) -> Dict[str, dict]:
        """获取所有Lane状态"""
        return {
            lane_id: {
                "status": lane.status.value,
                "current_task": lane.current_task,
                "queue_length": len(lane.queue)
            }
            for lane_id, lane in self.lanes.items()
        }
    
    async def _persist_lane_status(self, lane: Lane):
        """持久化Lane状态到Redis"""
        import json
        
        key = f"session:{self.session_key}:lanes"
        await self.redis.hset(
            key,
            lane.id,
            json.dumps({
                "status": lane.status.value,
                "current_task": lane.current_task,
                "queue": lane.queue
            })
        )
    
    async def _notify_task_start(self, task_id: str, lane_id: str):
        """通知任务开始执行"""
        # 发布到Redis，通知任务处理器
        await self.redis.publish(
            f"task:start:{task_id}",
            json.dumps({
                "task_id": task_id,
                "lane_id": lane_id,
                "session_key": self.session_key
            })
        )
```

### 5.4 Lane与任务路由

```python
# gateway/websocket/router.py
class RequestRouter:
    """请求路由器"""
    
    def __init__(self, redis_client, agent_dispatcher):
        self.redis = redis_client
        self.agent_dispatcher = agent_dispatcher
        self.lane_managers: Dict[str, LaneManager] = {}
    
    async def route_request(self, message: StandardMessage) -> dict:
        """
        路由请求到Agent
        
        流程:
        1. 获取或创建LaneManager
        2. 尝试获取空闲Lane
        3. 有空闲Lane → 立即分发
        4. 无空闲Lane → 加入队列
        """
        session_key = message.session_key
        
        # 获取或创建LaneManager
        if session_key not in self.lane_managers:
            self.lane_managers[session_key] = LaneManager(
                session_key, 
                self.redis
            )
        
        lane_manager = self.lane_managers[session_key]
        
        # 创建任务
        task_id = self._generate_task_id()
        
        # 尝试获取Lane
        lane_id = await lane_manager.acquire_lane()
        
        if lane_id:
            # 有空闲Lane，立即分发
            task = AgentTask(
                id=task_id,
                session_key=session_key,
                lane_id=lane_id,
                tenant_id=message.tenant_id,
                user_id=message.user_id,
                app_id=message.metadata.get("app_id", ""),
                message={"content": message.content}
            )
            
            await self.agent_dispatcher.dispatch(task)
            
            return {
                "task_id": task_id,
                "status": "dispatched",
                "lane_id": lane_id
            }
        else:
            # 无空闲Lane，加入队列
            lane_id = await lane_manager.enqueue_task(task_id)
            
            return {
                "task_id": task_id,
                "status": "queued",
                "lane_id": lane_id,
                "estimated_wait": await self._estimate_wait_time(session_key)
            }
    
    async def on_task_completed(self, task_id: str, lane_id: str, session_key: str):
        """任务完成回调"""
        lane_manager = self.lane_managers.get(session_key)
        if lane_manager:
            await lane_manager.release_lane(lane_id, task_id)

# agent/core/prompt_builder.py
from typing import List, Dict, Any, Optional
from datetime import datetime

class SystemPromptBuilder:
    """动态System Prompt构建器"""
    
    def __init__(self, agent_config: Dict[str, Any]):
        self.config = agent_config
        self.base_prompt = agent_config.get("base_prompt", "")
    
    async def build(
        self,
        context: Dict[str, Any],
        available_tools: List[Dict],
        skills: List[Dict],
        memory_summary: Optional[str] = None
    ) -> str:
        """
        构建完整的System Prompt
        
        组装顺序:
        1. 基础Prompt (Agent人格定义)
        2. 当前时间
        3. 可用工具列表
        4. 已安装Skills
        5. 记忆摘要
        6. 用户偏好
        """
        sections = []
        
        # 1. 基础Prompt
        if self.base_prompt:
            sections.append(self.base_prompt)
        
        # 2. 当前时间
        sections.append(f"\n当前时间: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}\n")
        
        # 3. 可用工具
        if available_tools:
            tool_descriptions = self._format_tools(available_tools)
            sections.append(f"\n## 可用工具\n\n{tool_descriptions}\n")
        
        # 4. 已安装Skills
        if skills:
            skill_descriptions = self._format_skills(skills)
            sections.append(f"\n## 已安装技能\n\n{skill_descriptions}\n")
        
        # 5. 记忆摘要
        if memory_summary:
            sections.append(f"\n## 相关记忆\n\n{memory_summary}\n")
        
        # 6. 用户偏好
        user_preferences = context.get("user_preferences", {})
        if user_preferences:
            pref_text = self._format_preferences(user_preferences)
            sections.append(f"\n## 用户偏好\n\n{pref_text}\n")
        
        return "\n".join(sections)
    
    def _format_tools(self, tools: List[Dict]) -> str:
        """格式化工具描述"""
        lines = []
        for tool in tools:
            lines.append(f"- **{tool['name']}**: {tool['description']}")
            if 'parameters' in tool:
                params = ", ".join(tool['parameters'].get('required', []))
                lines.append(f"  参数: {params}")
        return "\n".join(lines)
    
    def _format_skills(self, skills: List[Dict]) -> str:
        """格式化技能描述"""
        lines = []
        for skill in skills:
            lines.append(f"- **{skill['name']}**: {skill['description']}")
        return "\n".join(lines)
    
    def _format_preferences(self, preferences: Dict) -> str:
        """格式化用户偏好"""
        lines = []
        for key, value in preferences.items():
            lines.append(f"- {key}: {value}")
        return "\n".join(lines)

### 6.3 Agentic Loop实现

```python
# agent/core/loop.py
from typing import AsyncIterator, Dict, Any, Optional
from enum import Enum

class LoopStatus(Enum):
    CONTINUE = "continue"
    FINAL = "final"
    ERROR = "error"

class AgenticLoop:
    """
    智能体循环 - ReAct模式
    
    循环流程:
    1. 调用LLM
    2. 检查是否需要工具调用
    3. 执行工具
    4. 将结果加入上下文
    5. 回到步骤1，直到产生最终响应
    """
    
    MAX_ITERATIONS = 20  # 最大循环次数
    
    def __init__(
        self,
        llm_client,
        tool_executor,
        context_manager,
        prompt_builder: SystemPromptBuilder
    ):
        self.llm = llm_client
        self.tool_executor = tool_executor
        self.context = context_manager
        self.prompt_builder = prompt_builder
    
    async def run(
        self,
        user_message: str,
        session_key: str,
        stream: bool = True
    ) -> AsyncIterator[Dict[str, Any]]:
        """
        执行智能体循环
        
        Yields:
            流式响应块
        """
        iteration = 0
        
        while iteration < self.MAX_ITERATIONS:
            iteration += 1
            
            # 构建Prompt
            system_prompt = await self.prompt_builder.build(
                context=await self.context.get_context(session_key),
                available_tools=await self.tool_executor.get_available_tools(),
                skills=await self.context.get_skills(session_key)
            )
            
            # 调用LLM
            response = await self.llm.chat(
                system_prompt=system_prompt,
                messages=await self.context.get_messages(session_key),
                tools=await self.tool_executor.get_tool_definitions(),
                stream=stream
            )
            
            # 处理响应
            if stream:
                async for chunk in self._process_stream_response(response, session_key):
                    yield chunk
                    if chunk.get("status") == LoopStatus.FINAL:
                        return
            else:
                result = await self._process_response(response, session_key)
                yield result
                if result.get("status") == LoopStatus.FINAL:
                    return
        
        # 超过最大迭代次数
        yield {
            "status": LoopStatus.ERROR,
            "content": "达到最大迭代次数，请简化您的问题或联系支持。"
        }
    
    async def _process_stream_response(
        self,
        response: AsyncIterator,
        session_key: str
    ) -> AsyncIterator[Dict[str, Any]]:
        """处理流式响应"""
        tool_calls = []
        text_content = ""
        
        async for chunk in response:
            delta = chunk.get("delta", {})
            
            # 文本内容
            if "content" in delta:
                text_content += delta["content"]
                yield {
                    "status": LoopStatus.CONTINUE,
                    "type": "text_delta",
                    "content": delta["content"]
                }
            
            # 工具调用
            if "tool_calls" in delta:
                tool_calls.extend(delta["tool_calls"])
        
        # 处理工具调用
        if tool_calls:
            # 保存LLM响应到上下文
            await self.context.add_message(
                session_key,
                {"role": "assistant", "content": text_content, "tool_calls": tool_calls}
            )
            
            # 执行工具
            for tool_call in tool_calls:
                result = await self.tool_executor.execute(
                    tool_call["name"],
                    tool_call["arguments"]
                )
                
                # 保存工具结果
                await self.context.add_message(
                    session_key,
                    {
                        "role": "tool",
                        "tool_call_id": tool_call["id"],
                        "content": json.dumps(result)
                    }
                )
                
                yield {
                    "status": LoopStatus.CONTINUE,
                    "type": "tool_result",
                    "tool_name": tool_call["name"],
                    "result": result
                }
        else:
            # 无工具调用，这是最终响应
            await self.context.add_message(
                session_key,
                {"role": "assistant", "content": text_content}
            )
            
            yield {
                "status": LoopStatus.FINAL,
                "type": "final_response",
                "content": text_content
            }
```

```python
# agent/llm/resolver.py
from enum import Enum
from typing import Optional

class ModelTier(Enum):
    """模型层级"""
    FAST = "fast"           # 快速响应，成本低
    BALANCED = "balanced"   # 平衡性能和成本
    POWERFUL = "powerful"   # 最强能力，成本高

class ModelResolver:
    """模型选择器"""
    
    # 模型配置
    MODELS = {
        ModelTier.FAST: ["claude-3-haiku", "gpt-3.5-turbo", "deepseek-chat"],
        ModelTier.BALANCED: ["claude-3-sonnet", "gpt-4o-mini", "deepseek-reasoner"],
        ModelTier.POWERFUL: ["claude-3-opus", "gpt-4o", "deepseek-r1"]
    }
    
    def __init__(self, default_tier: ModelTier = ModelTier.BALANCED):
        self.default_tier = default_tier
        self.fallback_chain = [
            ModelTier.POWERFUL,
            ModelTier.BALANCED,
            ModelTier.FAST
        ]
    
    def resolve(
        self,
        task_complexity: str = "normal",
        user_preference: Optional[str] = None
    ) -> str:
        """
        根据任务复杂度选择模型
        
        Args:
            task_complexity: simple, normal, complex, deep_reasoning
            user_preference: 用户指定的模型偏好
        """
        if user_preference:
            return user_preference
        
        # 根据复杂度映射到模型层级
        tier_mapping = {
            "simple": ModelTier.FAST,
            "normal": ModelTier.BALANCED,
            "complex": ModelTier.POWERFUL,
            "deep_reasoning": ModelTier.POWERFUL
        }
        
        tier = tier_mapping.get(task_complexity, self.default_tier)
        
        # 返回该层级的第一个可用模型
        return self.MODELS[tier][0]
    
    async def get_fallback_model(self, failed_model: str) -> Optional[str]:
        """获取备用模型"""
        # 找到失败模型所在的层级
        for tier, models in self.MODELS.items():
            if failed_model in models:
                # 尝试同层级其他模型
                for model in models:
                    if model != failed_model:
                        return model
        
        # 尝试更低层级
        for tier in [ModelTier.BALANCED, ModelTier.FAST]:
            if self.MODELS[tier]:
                return self.MODELS[tier][0]
        
        return None
```

# agent/core/context.py
import json
from typing import List, Dict, Any, Optional
from datetime import datetime
import redis.asyncio as redis
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select

class ContextManager:
    """上下文管理器"""
    
    SHORT_TERM_TTL = 72 * 3600  # 72小时
    MAX_MESSAGES = 100          # 最大消息数
    MAX_TOKENS = 8000           # 最大Token数
    
    def __init__(
        self,
        redis_client: redis.Redis,
        db_session: AsyncSession,
        vector_store
    ):
        self.redis = redis_client
        self.db = db_session
        self.vector_store = vector_store
    
    async def get_messages(self, session_key: str) -> List[Dict]:
        """获取对话历史"""
        key = f"session:{session_key}:messages"
        data = await self.redis.lrange(key, 0, -1)
        
        messages = []
        for item in data:
            msg = json.loads(item)
            messages.append(msg)
        
        return messages
    
    async def add_message(self, session_key: str, message: Dict):
        """添加消息到历史"""
        key = f"session:{session_key}:messages"
        
        # 添加时间戳
        message["timestamp"] = datetime.utcnow().isoformat()
        
        # 添加到列表
        await self.redis.rpush(key, json.dumps(message))
        
        # 设置过期时间
        await self.redis.expire(key, self.SHORT_TERM_TTL)
        
        # 检查消息数，超过限制则压缩
        count = await self.redis.llen(key)
        if count > self.MAX_MESSAGES:
            await self._compress_messages(session_key)
    
    async def get_context(self, session_key: str) -> Dict[str, Any]:
        """获取完整上下文"""
        messages = await self.get_messages(session_key)
        
        # 获取用户偏好
        user_preferences = await self._get_user_preferences(session_key)
        
        # 获取相关记忆
        relevant_memories = await self._get_relevant_memories(session_key, messages)
        
        return {
            "messages": messages,
            "user_preferences": user_preferences,
            "relevant_memories": relevant_memories
        }

        压缩消息历史

        1. 保留最近20条消息
        2. 对早期消息生成摘要
        3. 标记重要消息保留
        """
        key = f"session:{session_key}:messages"
        
        # 获取所有消息
        all_messages = await self.get_messages(session_key)
        
        if len(all_messages) <= self.MAX_MESSAGES:
            return
        
        # 分割：保留最近的消息
        recent_messages = all_messages[-20:]
        old_messages = all_messages[:-20]
        
        # 找出重要消息（用户标记或包含工具调用）
        important_messages = [
            m for m in old_messages
            if m.get("important") or m.get("tool_calls")
        ]
        
        # 生成旧消息摘要
        summary = await self._generate_summary(old_messages)
        
        # 构建新的消息列表
        new_messages = []
        
        # 添加摘要作为系统消息
        if summary:
            new_messages.append({
                "role": "system",
                "content": f"[历史对话摘要]\n{summary}"
            })
        
        # 添加重要消息
        new_messages.extend(important_messages)
        
        # 添加最近消息
        new_messages.extend(recent_messages)
        
        # 替换Redis中的消息
        await self.redis.delete(key)
        for msg in new_messages:
            await self.redis.rpush(key, json.dumps(msg))
        
        await self.redis.expire(key, self.SHORT_TERM_TTL)

        return f"讨论了: {', '.join(list(topics)[:5])}"

        if not messages:

# agent/skills/registry.py
from typing import Dict, Type, List, Optional
from agent.skills.base import BaseSkill

class SkillRegistry:
    """技能注册表"""
    
    _skills: Dict[str, Type[BaseSkill]] = {}
    _instances: Dict[str, BaseSkill] = {}
    
    @classmethod
    def register(cls, skill_class: Type[BaseSkill]):
        """注册技能"""
        cls._skills[skill_class.name] = skill_class
    
    @classmethod
    def get_skill(cls, name: str, config: Optional[Dict] = None) -> Optional[BaseSkill]:
        """获取技能实例"""
        if name not in cls._skills:
            return None
        
        # 检查是否已有实例
        cache_key = f"{name}:{hash(str(config))}"
        if cache_key in cls._instances:
            return cls._instances[cache_key]
        
        # 创建新实例
        skill_class = cls._skills[name]
        instance = skill_class(config)
        cls._instances[cache_key] = instance
        
        return instance
    
    @classmethod
    def list_skills(cls) -> List[Dict[str, Any]]:
        """列出所有已注册技能"""
        return [
            {
                "name": name,
                "description": skill_class.description,
                "version": skill_class.version,
                "category": skill_class.category.value
            }
            for name, skill_class in cls._skills.items()
        ]
    
    @classmethod
    def get_tool_definitions(cls, skill_names: Optional[List[str]] = None) -> List[Dict]:
        """获取工具定义列表"""
        if skill_names is None:
            skill_names = list(cls._skills.keys())
        
        definitions = []
        for name in skill_names:
            skill = cls.get_skill(name)
            if skill:
                definitions.append(skill.get_tool_definition())
        
        return definitions

import asyncio
import json

class MCPClient:
    """
    Model Context Protocol 客户端
    
    用于连接MCP服务器，获取工具列表，执行工具调用
    """

        self.config = server_config
        self.transport = server_config.get("transport", "stdio")
        self.tools: List[Dict] = []
        self._process: Optional[asyncio.subprocess.Process] = None
    
    async def connect(self):
        """连接MCP服务器"""
        if self.transport == "stdio":
            await self._connect_stdio()
        elif self.transport == "sse":
            await self._connect_sse()
    
    async def _connect_stdio(self):
        """通过stdio连接"""
        command = self.config.get("command")
        args = self.config.get("args", [])
        
        self._process = await asyncio.create_subprocess_exec(
            command,
            *args,
            stdin=asyncio.subprocess.PIPE,
            stdout=asyncio.subprocess.PIPE,
            stderr=asyncio.subprocess.PIPE
        )
        
        # 发送初始化请求
        await self._send_request("initialize", {
            "protocolVersion": "2024-11-05",
            "capabilities": {}
        })
        
        # 获取工具列表
        tools_response = await self._send_request("tools/list", {})
        self.tools = tools_response.get("tools", [])
    
    async def _send_request(self, method: str, params: Dict) -> Dict:
        """发送请求"""
        request = {
            "jsonrpc": "2.0",
            "id": self._next_id(),
            "method": method,
            "params": params
        }
        
        self._process.stdin.write((json.dumps(request) + "\n").encode())
        await self._process.stdin.drain()
        
        response_line = await self._process.stdout.readline()
        return json.loads(response_line)
    
    async def execute_tool(self, tool_name: str, arguments: Dict) -> Dict[str, Any]:
        """执行工具"""
        response = await self._send_request("tools/call", {
            "name": tool_name,
            "arguments": arguments
        })
        
        return response.get("result", {})
    
    def get_tools(self) -> List[Dict]:
        """获取可用工具列表"""
        return self.tools
    
    def _next_id(self) -> int:
        """生成下一个请求ID"""
        if not hasattr(self, "_id_counter"):
            self._id_counter = 0
        self._id_counter += 1
        return self._id_counter

## 9. Agent分发系统

Agent分发系统负责任务队列管理、Agent实例池维护和负载均衡调度。采用Redis作为分布式任务队列，结合状态机管理Agent生命周期，实现高可用和高性能的Agent任务处理。

### 9.1 任务队列设计

#### 9.1.1 Redis优先级队列实现

任务队列采用Redis有序集合(Sorted Set)实现优先级队列，支持多级优先级和延迟任务：

```python
# agent/distribution/queue.py
import json
import time
from typing import Optional, List
from dataclasses import dataclass, asdict
from enum import Enum
import redis.asyncio as redis

class TaskPriority(Enum):
    CRITICAL = 0     # 关键任务，立即执行
    HIGH = 1         # 高优先级
    NORMAL = 2       # 普通优先级
    LOW = 3          # 低优先级
    BACKGROUND = 4   # 后台任务

class TaskStatus(Enum):
    PENDING = "pending"           # 等待中
    QUEUED = "queued"             # 已入队
    ASSIGNED = "assigned"         # 已分配Agent
    RUNNING = "running"           # 执行中
    COMPLETED = "completed"       # 已完成
    FAILED = "failed"             # 失败
    CANCELLED = "cancelled"       # 已取消
    TIMEOUT = "timeout"           # 超时

@dataclass
class AgentTask:
    """Agent任务定义"""
    id: str
    session_key: str
    lane_id: str
    tenant_id: str
    user_id: str
    app_id: str
    message: dict
    priority: int = 2  # TaskPriority.NORMAL
    status: str = "pending"
    created_at: float = 0
    assigned_at: Optional[float] = None
    started_at: Optional[float] = None
    completed_at: Optional[float] = None
    agent_id: Optional[str] = None
    retry_count: int = 0
    max_retries: int = 3
    timeout_seconds: int = 300
    
    def __post_init__(self):
        if self.created_at == 0:
            self.created_at = time.time()
    
    def to_json(self) -> str:
        return json.dumps(asdict(self), default=str)
    
    @classmethod
    def from_json(cls, data: str) -> "AgentTask":
        return cls(**json.loads(data))

class RedisPriorityQueue:
    """Redis优先级队列实现"""
    
    def __init__(self, redis_client: redis.Redis, queue_name: str):
        self.redis = redis_client
        self.queue_key = f"queue:{queue_name}"
        self.processing_key = f"queue:{queue_name}:processing"
        self.dead_letter_key = f"queue:{queue_name}:dead_letter"
    
    async def enqueue(self, task: AgentTask) -> bool:
        """将任务加入队列
        
        Args:
            task: Agent任务
            
        Returns:
            bool: 是否成功入队
        """
        # 计算优先级分数 (越小优先级越高)
        # 公式: priority * 1000000000 + timestamp
        # 确保同优先级下FIFO
        score = task.priority * 1_000_000_000 + task.created_at
        
        task.status = TaskStatus.QUEUED.value
        
        # 添加到有序集合
        await self.redis.zadd(
            self.queue_key,
            {task.to_json(): score}
        )
        
        # 更新任务状态Hash
        await self.redis.hset(
            f"task:{task.id}",
            mapping={
                "status": task.status,
                "tenant_id": task.tenant_id,
                "created_at": task.created_at,
                "queue": self.queue_key
            }
        )
        
        return True
    
    async def dequeue(self, timeout: int = 5) -> Optional[AgentTask]:
        """从队列取出优先级最高的任务
        
        Args:
            timeout: 阻塞超时时间(秒)
            
        Returns:
            AgentTask或None
        """
        # 使用zrange获取优先级最高的任务
        results = await self.redis.zrange(
            self.queue_key,
            0, 0,  # 取第一个
            withscores=False
        )
        
        if not results:
            return None
        
        task_data = results[0]
        
        # 从队列移除
        removed = await self.redis.zrem(self.queue_key, task_data)
        if removed == 0:
            # 任务已被其他消费者取走
            return None
        
        task = AgentTask.from_json(task_data)
        task.status = TaskStatus.ASSIGNED.value
        task.assigned_at = time.time()
        
        # 加入处理中集合
        await self.redis.zadd(
            self.processing_key,
            {task.id: task.assigned_at + task.timeout_seconds}
        )
        
        return task
    
    async def complete(self, task_id: str, success: bool = True):
        """标记任务完成
        
        Args:
            task_id: 任务ID
            success: 是否成功
        """
        # 从处理中集合移除
        await self.redis.zrem(self.processing_key, task_id)
        
        # 更新任务状态
        status = TaskStatus.COMPLETED.value if success else TaskStatus.FAILED.value
        await self.redis.hset(
            f"task:{task_id}",
            "status",
            status
        )
        await self.redis.hset(
            f"task:{task_id}",
            "completed_at",
            time.time()
        )
    
    async def requeue(self, task: AgentTask) -> bool:
        """重新入队(用于重试)"""
        if task.retry_count >= task.max_retries:
            # 超过最大重试次数，加入死信队列
            await self.redis.lpush(
                self.dead_letter_key,
                task.to_json()
            )
            return False
        
        task.retry_count += 1
        task.status = TaskStatus.PENDING.value
        task.assigned_at = None
        task.agent_id = None
        
        # 延迟重试，指数退避
        delay = min(2 ** task.retry_count, 60)  # 最大延迟60秒
        task.created_at = time.time() + delay
        
        return await self.enqueue(task)
    
    async def get_queue_length(self) -> int:
        """获取队列长度"""
        return await self.redis.zcard(self.queue_key)
    
    async def get_processing_count(self) -> int:
        """获取处理中任务数"""
        return await self.redis.zcard(self.processing_key)
    
    async def recover_timeout_tasks(self) -> List[AgentTask]:
        """恢复超时任务"""
        now = time.time()
        
        # 获取所有超时的任务
        timeout_tasks = await self.redis.zrangebyscore(
            self.processing_key,
            0,
            now
        )
        
        recovered = []
        for task_id in timeout_tasks:
            # 从处理中移除
            await self.redis.zrem(self.processing_key, task_id)
            
            # 获取任务详情
            task_data = await self.redis.hgetall(f"task:{task_id}")
            if task_data:
                task = AgentTask(
                    id=task_id,
                    session_key=task_data.get(b'session_key', b'').decode(),
                    lane_id=task_data.get(b'lane_id', b'').decode(),
                    tenant_id=task_data.get(b'tenant_id', b'').decode(),
                    user_id=task_data.get(b'user_id', b'').decode(),
                    app_id=task_data.get(b'app_id', b'').decode(),
                    message={}
                )
                recovered.append(task)
        
        return recovered

class MultiTenantQueueManager:
    """多租户队列管理器"""
    
    def __init__(self, redis_client: redis.Redis):
        self.redis = redis_client
        self.queues: dict[str, RedisPriorityQueue] = {}
    
    def get_queue(self, tenant_id: str) -> RedisPriorityQueue:
        """获取租户专属队列"""
        if tenant_id not in self.queues:
            self.queues[tenant_id] = RedisPriorityQueue(
                self.redis,
                f"tenant:{tenant_id}"
            )
        return self.queues[tenant_id]
    
    async def get_global_stats(self) -> dict:
        """获取全局队列统计"""
        stats = {
            "total_pending": 0,
            "total_processing": 0,
            "tenant_stats": {}
        }
        
        # 扫描所有租户队列
        cursor = 0
        pattern = "queue:tenant:*"
        
        while True:
            cursor, keys = await self.redis.scan(cursor, match=pattern)
            for key in keys:
                tenant_id = key.decode().split(":")[2]
                queue = self.get_queue(tenant_id)
                
                pending = await queue.get_queue_length()
                processing = await queue.get_processing_count()
                
                stats["tenant_stats"][tenant_id] = {
                    "pending": pending,
                    "processing": processing
                }
                stats["total_pending"] += pending
                stats["total_processing"] += processing
            
            if cursor == 0:
                break
        
        return stats

#### 9.2.1 AgentState状态机

```python
# agent/distribution/state_machine.py
from enum import Enum, auto
from typing import Optional, Callable
from dataclasses import dataclass
from datetime import datetime
import asyncio

class AgentState(Enum):
    """Agent实例状态枚举"""
    INITIALIZING = "initializing"    # 初始化中
    IDLE = "idle"                    # 空闲
    BUSY = "busy"                    # 忙碌(处理任务)
    PAUSED = "paused"                # 暂停(维护模式)
    UNHEALTHY = "unhealthy"          # 不健康
    SHUTTING_DOWN = "shutting_down"  # 关闭中
    TERMINATED = "terminated"        # 已终止

@dataclass
class AgentInstance:
    """Agent实例信息"""
    id: str
    host: str
    port: int
    state: AgentState
    tenant_id: Optional[str]  # None表示共享池
    current_task: Optional[str]  # 当前处理的任务ID
    capabilities: list[str]   # 支持的技能列表
    load_factor: float        # 负载因子(0-1)
    last_heartbeat: datetime
    created_at: datetime
    task_count: int = 0       # 累计处理任务数
    error_count: int = 0      # 累计错误数

class StateTransition:
    """状态转换定义"""
    
    # 允许的状态转换
    ALLOWED_TRANSITIONS = {
        AgentState.INITIALIZING: [AgentState.IDLE, AgentState.UNHEALTHY],
        AgentState.IDLE: [AgentState.BUSY, AgentState.PAUSED, AgentState.SHUTTING_DOWN, AgentState.UNHEALTHY],
        AgentState.BUSY: [AgentState.IDLE, AgentState.UNHEALTHY, AgentState.SHUTTING_DOWN],
        AgentState.PAUSED: [AgentState.IDLE, AgentState.SHUTTING_DOWN],
        AgentState.UNHEALTHY: [AgentState.IDLE, AgentState.SHUTTING_DOWN],
        AgentState.SHUTTING_DOWN: [AgentState.TERMINATED],
        AgentState.TERMINATED: []
    }
    
    @classmethod
    def can_transition(cls, from_state: AgentState, to_state: AgentState) -> bool:
        """检查状态转换是否允许"""
        return to_state in cls.ALLOWED_TRANSITIONS.get(from_state, [])

class AgentStateMachine:
    """Agent状态机管理器"""
    
    def __init__(self, agent_id: str):
        self.agent_id = agent_id
        self.state = AgentState.INITIALIZING
        self.state_history: list[tuple[datetime, AgentState, Optional[str]]] = []
        self._transition_handlers: dict[AgentState, list[Callable]] = {}
        self._lock = asyncio.Lock()
    
    async def transition_to(
        self, 
        new_state: AgentState, 
        reason: Optional[str] = None
    ) -> bool:
        """执行状态转换
        
        Args:
            new_state: 目标状态
            reason: 转换原因
            
        Returns:
            bool: 转换是否成功
        """
        async with self._lock:
            if not StateTransition.can_transition(self.state, new_state):
                return False
            
            old_state = self.state
            self.state = new_state
            
            # 记录历史
            self.state_history.append((
                datetime.utcnow(),
                new_state,
                reason
            ))
            
            # 触发回调
            await self._trigger_handlers(new_state, old_state, reason)
            
            return True
    
    def on_transition(self, state: AgentState, handler: Callable):
        """注册状态转换回调"""
        if state not in self._transition_handlers:
            self._transition_handlers[state] = []
        self._transition_handlers[state].append(handler)
    
    async def _trigger_handlers(
        self, 
        new_state: AgentState, 
        old_state: AgentState,
        reason: Optional[str]
    ):
        """触发状态回调"""
        handlers = self._transition_handlers.get(new_state, [])
        for handler in handlers:
            try:
                await handler(self.agent_id, old_state, new_state, reason)
            except Exception as e:
                print(f"State handler error: {e}")
    
    def get_state_duration(self) -> float:
        """获取当前状态持续时间(秒)"""
        if not self.state_history:
            return 0
        last_change = self.state_history[-1][0]
        return (datetime.utcnow() - last_change).total_seconds()

```python
# agent/distribution/pool.py
import asyncio
from typing import Optional, List, Callable
from datetime import datetime, timedelta
import redis.asyncio as redis

from .state_machine import AgentState, AgentInstance, AgentStateMachine

class AgentPoolManager:
    """Agent池管理器
    
    管理Agent实例的生命周期，包括:
    - 实例注册与发现
    - 健康检查
    - 负载均衡
    - 自动扩缩容
    """
    
    def __init__(
        self,
        redis_client: redis.Redis,
        max_agents: int = 100,
        min_agents: int = 2,
        health_check_interval: int = 30
    ):
        self.redis = redis_client
        self.max_agents = max_agents
        self.min_agents = min_agents
        self.health_check_interval = health_check_interval
        
        self.agents: dict[str, AgentInstance] = {}
        self.state_machines: dict[str, AgentStateMachine] = {}
        self._health_check_task: Optional[asyncio.Task] = None
        self._scaling_task: Optional[asyncio.Task] = None
    
    async def start(self):
        """启动管理器"""
        self._health_check_task = asyncio.create_task(self._health_check_loop())
        self._scaling_task = asyncio.create_task(self._scaling_loop())
    
    async def stop(self):
        """停止管理器"""
        if self._health_check_task:
            self._health_check_task.cancel()
        if self._scaling_task:
            self._scaling_task.cancel()
        
        # 优雅关闭所有Agent
        await self._graceful_shutdown()
    
    async def register_agent(
        self,
        agent_id: str,
        host: str,
        port: int,
        tenant_id: Optional[str] = None,
        capabilities: Optional[List[str]] = None
    ) -> bool:
        """注册新Agent实例
        
        Args:
            agent_id: Agent唯一标识
            host: 主机地址
            port: 服务端口
            tenant_id: 租户ID(可选，None表示共享池)
            capabilities: 支持的能力列表
            
        Returns:
            bool: 注册是否成功
        """
        if len(self.agents) >= self.max_agents:
            return False
        
        agent = AgentInstance(
            id=agent_id,
            host=host,
            port=port,
            state=AgentState.INITIALIZING,
            tenant_id=tenant_id,
            current_task=None,
            capabilities=capabilities or [],
            load_factor=0.0,
            last_heartbeat=datetime.utcnow(),
            created_at=datetime.utcnow()
        )
        
        self.agents[agent_id] = agent
        
        # 初始化状态机
        sm = AgentStateMachine(agent_id)
        sm.on_transition(AgentState.IDLE, self._on_agent_ready)
        sm.on_transition(AgentState.UNHEALTHY, self._on_agent_unhealthy)
        self.state_machines[agent_id] = sm
        
        # 持久化到Redis
        await self._persist_agent(agent)
        
        # 执行初始化转换
        await sm.transition_to(AgentState.IDLE, "Initialization complete")
        agent.state = AgentState.IDLE
        
        return True
    
    async def unregister_agent(self, agent_id: str, force: bool = False) -> bool:
        """注销Agent实例"""
        if agent_id not in self.agents:
            return False
        
        agent = self.agents[agent_id]
        sm = self.state_machines[agent_id]
        
        if not force and agent.state == AgentState.BUSY:
            # 等待任务完成
            await sm.transition_to(AgentState.SHUTTING_DOWN, "Waiting for task completion")
            # 设置超时
            await asyncio.wait_for(
                self._wait_for_idle(agent_id),
                timeout=60
            )
        
        # 执行状态转换
        await sm.transition_to(AgentState.TERMINATED, "Agent unregistered")
        
        # 清理
        del self.agents[agent_id]
        del self.state_machines[agent_id]
        await self.redis.delete(f"agent:{agent_id}")
        
        return True
    
    async def assign_task(self, task) -> Optional[str]:
        """为任务分配Agent
        
        Args:
            task: Agent任务
            
        Returns:
            Optional[str]: 分配的Agent ID，无可用Agent时返回None
        """
        # 获取候选Agent
        candidates = self._get_candidates(task.tenant_id, task.message.get('required_capabilities', []))
        
        if not candidates:
            return None
        
        # 选择负载最低的Agent
        selected = min(candidates, key=lambda a: a.load_factor)
        
        # 更新状态
        sm = self.state_machines[selected.id]
        success = await sm.transition_to(AgentState.BUSY, f"Assigned task {task.id}")
        
        if success:
            selected.state = AgentState.BUSY
            selected.current_task = task.id
            selected.load_factor = min(1.0, selected.load_factor + 0.3)
            await self._persist_agent(selected)
            return selected.id
        
        return None
    
    async def release_agent(self, agent_id: str, task_completed: bool = True):
        """释放Agent(任务完成)"""
        if agent_id not in self.agents:
            return
        
        agent = self.agents[agent_id]
        sm = self.state_machines[agent_id]
        
        # 更新统计
        agent.task_count += 1
        agent.current_task = None
        agent.load_factor = max(0.0, agent.load_factor - 0.3)
        
        if not task_completed:
            agent.error_count += 1
        
        # 状态转换
        await sm.transition_to(AgentState.IDLE, "Task completed")
        agent.state = AgentState.IDLE
        
        await self._persist_agent(agent)
    
    def _get_candidates(
        self,
        tenant_id: str,
        required_capabilities: List[str]
    ) -> List[AgentInstance]:
        """获取候选Agent列表"""
        candidates = []
        
        for agent in self.agents.values():
            # 状态检查
            if agent.state != AgentState.IDLE:
                continue
            
            # 租户检查
            if agent.tenant_id is not None and agent.tenant_id != tenant_id:
                continue
            
            # 能力检查
            if required_capabilities:
                if not all(cap in agent.capabilities for cap in required_capabilities):
                    continue
            
            candidates.append(agent)
        
        return candidates
    
    async def _health_check_loop(self):
        """健康检查循环"""
        while True:
            try:
                await asyncio.sleep(self.health_check_interval)
                await self._check_all_agents()
            except asyncio.CancelledError:
                break
            except Exception as e:
                print(f"Health check error: {e}")
    
    async def _check_all_agents(self):
        """检查所有Agent健康状态"""
        now = datetime.utcnow()
        timeout_threshold = now - timedelta(seconds=self.health_check_interval * 2)
        
        for agent_id, agent in list(self.agents.items()):
            # 检查心跳超时
            if agent.last_heartbeat < timeout_threshold:
                sm = self.state_machines[agent_id]
                await sm.transition_to(AgentState.UNHEALTHY, "Heartbeat timeout")
                agent.state = AgentState.UNHEALTHY
    
    async def _scaling_loop(self):
        """自动扩缩容循环"""
        while True:
            try:
                await asyncio.sleep(60)  # 每分钟检查一次
                await self._evaluate_scaling()
            except asyncio.CancelledError:
                break
            except Exception as e:
                print(f"Scaling error: {e}")
    
    async def _evaluate_scaling(self):
        """评估是否需要扩缩容"""
        # 获取队列长度
        queue_length = await self.redis.llen("agent:task_queue:global")
        
        # 获取忙碌Agent数量
        busy_count = sum(1 for a in self.agents.values() if a.state == AgentState.BUSY)
        idle_count = sum(1 for a in self.agents.values() if a.state == AgentState.IDLE)
        total_count = len(self.agents)
        
        # 扩容判断
        if queue_length > idle_count * 2 and total_count < self.max_agents:
            await self._scale_up()
        
        # 缩容判断
        elif idle_count > self.min_agents and queue_length == 0:
            await self._scale_down()
    
    async def _scale_up(self):
        """扩容 - 创建新Agent实例"""
        # 触发扩容事件，由外部编排器(K8s)处理
        await self.redis.publish("agent:scale_up", "1")
    
    async def _scale_down(self):
        """缩容 - 移除空闲Agent"""
        # 找到最空闲的Agent
        idle_agents = [a for a in self.agents.values() if a.state == AgentState.IDLE]
        if len(idle_agents) <= self.min_agents:
            return
        
        # 按创建时间排序，优先移除最新的
        to_remove = sorted(idle_agents, key=lambda a: a.created_at, reverse=True)[0]
        await self.unregister_agent(to_remove.id)
    
    async def _persist_agent(self, agent: AgentInstance):
        """持久化Agent信息到Redis"""
        await self.redis.hset(
            f"agent:{agent.id}",
            mapping={
                "host": agent.host,
                "port": agent.port,
                "state": agent.state.value,
                "tenant_id": agent.tenant_id or "shared",
                "current_task": agent.current_task or "",
                "capabilities": ",".join(agent.capabilities),
                "load_factor": str(agent.load_factor),
                "last_heartbeat": agent.last_heartbeat.isoformat(),
                "task_count": agent.task_count,
                "error_count": agent.error_count
            }
        )
    
    async def _on_agent_ready(self, agent_id: str, old_state, new_state, reason):
        """Agent就绪回调"""
        await self.redis.sadd("agent:pool:ready", agent_id)
    
    async def _on_agent_unhealthy(self, agent_id: str, old_state, new_state, reason):
        """Agent不健康回调"""
        await self.redis.srem("agent:pool:ready", agent_id)
        # 尝试重启或替换
        asyncio.create_task(self._handle_unhealthy_agent(agent_id))
    
    async def _handle_unhealthy_agent(self, agent_id: str):
        """处理不健康Agent"""
        # 尝试优雅关闭
        await self.unregister_agent(agent_id, force=False)
        # 触发重新创建
        await self.redis.publish("agent:recreate", agent_id)
    
    async def _graceful_shutdown(self):
        """优雅关闭所有Agent"""
        tasks = []
        for agent_id in list(self.agents.keys()):
            tasks.append(self.unregister_agent(agent_id))
        await asyncio.gather(*tasks, return_exceptions=True)
    
    async def _wait_for_idle(self, agent_id: str):
        """等待Agent变为空闲"""
        while self.agents.get(agent_id, AgentInstance("", "", 0, AgentState.TERMINATED, None, None, [], 0, datetime.utcnow(), datetime.utcnow())).state == AgentState.BUSY:
            await asyncio.sleep(1)
```

#### 9.3.1 策略接口与实现

```python
# agent/distribution/load_balancer.py
from abc import ABC, abstractmethod
from typing import List, Optional
from dataclasses import dataclass
import random

from .state_machine import AgentInstance

@dataclass
class LoadBalancerConfig:
    """负载均衡配置"""
    strategy: str = "round_robin"
    weights: Optional[dict[str, int]] = None  # 加权策略使用
    health_check_enabled: bool = True

class LoadBalancerStrategy(ABC):
    """负载均衡策略基类"""
    
    @abstractmethod
    def select(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        """从候选Agent中选择一个"""
        pass
    
    @abstractmethod
    def get_name(self) -> str:
        """策略名称"""
        pass

class RoundRobinStrategy(LoadBalancerStrategy):
    """轮询策略"""
    
    def __init__(self):
        self._index = 0
    
    def select(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        if not agents:
            return None
        
        # 简单轮询
        selected = agents[self._index % len(agents)]
        self._index = (self._index + 1) % len(agents)
        return selected
    
    def get_name(self) -> str:
        return "round_robin"

class LeastConnectionsStrategy(LoadBalancerStrategy):
    """最少连接数策略
    
    选择当前处理任务最少的Agent
    """
    
    def select(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        if not agents:
            return None
        
        # 选择load_factor最低的Agent
        return min(agents, key=lambda a: a.load_factor)
    
    def get_name(self) -> str:
        return "least_connections"

class WeightedRoundRobinStrategy(LoadBalancerStrategy):
    """加权轮询策略
    
    根据配置的权重分配请求
    """
    
    def __init__(self, weights: dict[str, int]):
        self.weights = weights
        self._current_index = 0
        self._current_weight = 0
        self._max_weight = max(weights.values()) if weights else 1
    
    def select(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        if not agents:
            return None
        
        # 获取有权重的Agent
        weighted_agents = []
        for agent in agents:
            weight = self.weights.get(agent.id, 1)
            weighted_agents.extend([agent] * weight)
        
        if not weighted_agents:
            return agents[0]
        
        selected = weighted_agents[self._current_index % len(weighted_agents)]
        self._current_index += 1
        return selected
    
    def get_name(self) -> str:
        return "weighted_round_robin"

class CapabilityMatchingStrategy(LoadBalancerStrategy):
    """能力匹配策略
    
    根据任务所需能力选择最合适的Agent
    """
    
    def select(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        if not agents:
            return None
        
        required_caps = task.message.get('required_capabilities', [])
        if not required_caps:
            # 无特殊要求，使用最少连接
            return min(agents, key=lambda a: a.load_factor)
        
        # 计算每个Agent的匹配分数
        scored_agents = []
        for agent in agents:
            match_count = sum(1 for cap in required_caps if cap in agent.capabilities)
            score = match_count / len(required_caps)
            scored_agents.append((agent, score))
        
        # 按匹配分数排序，分数相同按负载排序
        scored_agents.sort(key=lambda x: (-x[1], x[0].load_factor))
        
        return scored_agents[0][0] if scored_agents else None
    
    def get_name(self) -> str:
        return "capability_matching"

class LoadBalancer:
    """负载均衡器"""
    
    STRATEGIES = {
        "round_robin": RoundRobinStrategy,
        "least_connections": LeastConnectionsStrategy,
        "weighted_round_robin": WeightedRoundRobinStrategy,
        "capability_matching": CapabilityMatchingStrategy
    }
    
    def __init__(self, config: LoadBalancerConfig):
        self.config = config
        self.strategy = self._create_strategy(config)
    
    def _create_strategy(self, config: LoadBalancerConfig) -> LoadBalancerStrategy:
        """创建策略实例"""
        strategy_class = self.STRATEGIES.get(config.strategy)
        if not strategy_class:
            raise ValueError(f"Unknown strategy: {config.strategy}")
        
        if config.strategy == "weighted_round_robin":
            return strategy_class(config.weights or {})
        
        return strategy_class()
    
    def select_agent(
        self,
        agents: List[AgentInstance],
        task
    ) -> Optional[AgentInstance]:
        """选择Agent"""
        # 过滤掉不健康的Agent
        healthy_agents = [
            a for a in agents
            if a.state.value in ["idle", "busy"]  # 允许busy状态的Agent作为备选
        ]
        
        if not healthy_agents:
            return None
        
        # 优先选择idle状态的Agent
        idle_agents = [a for a in healthy_agents if a.state.value == "idle"]
        if idle_agents:
            return self.strategy.select(idle_agents, task)
        
        # 如果没有idle的，选择负载最低的busy Agent
        return min(healthy_agents, key=lambda a: a.load_factor)
    
    def update_strategy(self, config: LoadBalancerConfig):
        """更新负载均衡策略"""
        self.config = config
        self.strategy = self._create_strategy(config)

| 策略 | 适用场景 | 优点 | 缺点 |

from contextlib import asynccontextmanager
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import text

class TenantSession:
    """支持租户隔离的数据库会话"""
    
    def __init__(self, session: AsyncSession):
        self.session = session
        self.current_tenant: Optional[str] = None
    
    async def set_tenant_context(self, tenant_id: str):
        """设置当前租户上下文(触发RLS)"""
        self.current_tenant = tenant_id
        await self.session.execute(
            text("SET LOCAL app.current_tenant = :tenant_id"),
            {"tenant_id": tenant_id}
        )
    
    async def clear_tenant_context(self):
        """清除租户上下文"""
        self.current_tenant = None
        await self.session.execute(
            text("RESET app.current_tenant")
        )
    
    @asynccontextmanager
    async def tenant_scope(self, tenant_id: str):
        """租户上下文管理器"""
        await self.set_tenant_context(tenant_id)
        try:
            yield self.session
        finally:
            await self.clear_tenant_context()

# 依赖注入使用示例

        tenant_session = TenantSession(session)
        await tenant_session.set_tenant_context(tenant_id)

```python
# core/quota.py
from enum import Enum
from dataclasses import dataclass
from typing import Dict, Optional
from datetime import datetime, timedelta
import redis.asyncio as redis

class QuotaResource(Enum):
    """可配额资源类型"""
    AGENT_CALLS = "agent_calls"           # Agent调用次数
    CONVERSATIONS = "conversations"        # 对话数
    MESSAGES = "messages"                  # 消息数
    TOKENS = "tokens"                      # LLM Token数
    STORAGE_MB = "storage_mb"              # 存储空间(MB)
    USERS = "users"                        # 用户数
    APPLICATIONS = "applications"          # 应用数
    API_CALLS = "api_calls"               # API调用次数

class QuotaPeriod(Enum):
    """配额周期"""
    MINUTE = "minute"
    HOUR = "hour"
    DAY = "day"
    WEEK = "week"
    MONTH = "month"
    YEAR = "year"

@dataclass
class QuotaDefinition:
    """配额定义"""
    resource: QuotaResource
    limit: int
    period: QuotaPeriod
    burst: int = 0  # 突发容量
    
@dataclass
class QuotaUsage:
    """配额使用情况"""
    resource: QuotaResource
    used: int
    limit: int
    reset_at: datetime
    remaining: int
    exceeded: bool

class QuotaManager:
    """租户配额管理器"""
    
    # 默认配额配置
    DEFAULT_QUOTAS = {
        "free": {
            QuotaResource.AGENT_CALLS: QuotaDefinition(QuotaResource.AGENT_CALLS, 1000, QuotaPeriod.MONTH),
            QuotaResource.CONVERSATIONS: QuotaDefinition(QuotaResource.CONVERSATIONS, 100, QuotaPeriod.MONTH),
            QuotaResource.MESSAGES: QuotaDefinition(QuotaResource.MESSAGES, 1000, QuotaPeriod.MONTH),
            QuotaResource.TOKENS: QuotaDefinition(QuotaResource.TOKENS, 100000, QuotaPeriod.MONTH),
            QuotaResource.STORAGE_MB: QuotaDefinition(QuotaResource.STORAGE_MB, 100, QuotaPeriod.MONTH),
            QuotaResource.USERS: QuotaDefinition(QuotaResource.USERS, 5, QuotaPeriod.MONTH),
            QuotaResource.APPLICATIONS: QuotaDefinition(QuotaResource.APPLICATIONS, 3, QuotaPeriod.MONTH),
            QuotaResource.API_CALLS: QuotaDefinition(QuotaResource.API_CALLS, 10000, QuotaPeriod.MONTH),
        },
        "pro": {
            QuotaResource.AGENT_CALLS: QuotaDefinition(QuotaResource.AGENT_CALLS, 10000, QuotaPeriod.MONTH),
            QuotaResource.CONVERSATIONS: QuotaDefinition(QuotaResource.CONVERSATIONS, 1000, QuotaPeriod.MONTH),
            QuotaResource.MESSAGES: QuotaDefinition(QuotaResource.MESSAGES, 10000, QuotaPeriod.MONTH),
            QuotaResource.TOKENS: QuotaDefinition(QuotaResource.TOKENS, 1000000, QuotaPeriod.MONTH),
            QuotaResource.STORAGE_MB: QuotaDefinition(QuotaResource.STORAGE_MB, 1000, QuotaPeriod.MONTH),
            QuotaResource.USERS: QuotaDefinition(QuotaResource.USERS, 50, QuotaPeriod.MONTH),
            QuotaResource.APPLICATIONS: QuotaDefinition(QuotaResource.APPLICATIONS, 20, QuotaPeriod.MONTH),
            QuotaResource.API_CALLS: QuotaDefinition(QuotaResource.API_CALLS, 100000, QuotaPeriod.MONTH),
        },
        "enterprise": {
            QuotaResource.AGENT_CALLS: QuotaDefinition(QuotaResource.AGENT_CALLS, -1, QuotaPeriod.MONTH),  # 无限制
            QuotaResource.CONVERSATIONS: QuotaDefinition(QuotaResource.CONVERSATIONS, -1, QuotaPeriod.MONTH),
            QuotaResource.MESSAGES: QuotaDefinition(QuotaResource.MESSAGES, -1, QuotaPeriod.MONTH),
            QuotaResource.TOKENS: QuotaDefinition(QuotaResource.TOKENS, -1, QuotaPeriod.MONTH),
            QuotaResource.STORAGE_MB: QuotaDefinition(QuotaResource.STORAGE_MB, 10000, QuotaPeriod.MONTH),
            QuotaResource.USERS: QuotaDefinition(QuotaResource.USERS, -1, QuotaPeriod.MONTH),
            QuotaResource.APPLICATIONS: QuotaDefinition(QuotaResource.APPLICATIONS, 100, QuotaPeriod.MONTH),
            QuotaResource.API_CALLS: QuotaDefinition(QuotaResource.API_CALLS, -1, QuotaPeriod.MONTH),
        }
    }
    
    def __init__(self, redis_client: redis.Redis):
        self.redis = redis_client
    
    async def get_quota_usage(
        self,
        tenant_id: str,
        resource: QuotaResource,
        plan: str = "free"
    ) -> QuotaUsage:
        """获取资源配额使用情况"""
        
        # 获取配额定义
        definition = self.DEFAULT_QUOTAS.get(plan, {}).get(resource)
        if not definition:
            # 无限制
            return QuotaUsage(
                resource=resource,
                used=0,
                limit=-1,
                reset_at=datetime.utcnow(),
                remaining=-1,
                exceeded=False
            )
        
        # 获取当前使用量
        key = f"quota:{tenant_id}:{resource.value}:{definition.period.value}"
        used = int(await self.redis.get(key) or 0)
        
        # 计算重置时间
        reset_at = self._get_reset_time(definition.period)
        
        # 计算剩余
        remaining = definition.limit - used if definition.limit > 0 else -1
        
        return QuotaUsage(
            resource=resource,
            used=used,
            limit=definition.limit,
            reset_at=reset_at,
            remaining=remaining,
            exceeded=definition.limit > 0 and used >= definition.limit
        )
    
    async def check_quota(
        self,
        tenant_id: str,
        resource: QuotaResource,
        amount: int = 1,
        plan: str = "free"
    ) -> bool:
        """检查是否超出配额
        
        Returns:
            bool: True表示未超出配额，可以继续
        """
        usage = await self.get_quota_usage(tenant_id, resource, plan)
        
        if usage.limit == -1:
            return True  # 无限制
        
        return (usage.used + amount) <= usage.limit
    
    async def consume_quota(
        self,
        tenant_id: str,
        resource: QuotaResource,
        amount: int = 1,
        plan: str = "free"
    ) -> QuotaUsage:
        """消耗配额
        
        Returns:
            QuotaUsage: 更新后的配额使用情况
        """
        definition = self.DEFAULT_QUOTAS.get(plan, {}).get(resource)
        if not definition:
            return await self.get_quota_usage(tenant_id, resource, plan)
        
        key = f"quota:{tenant_id}:{resource.value}:{definition.period.value}"
        
        # 原子增加使用量
        new_used = await self.redis.incrby(key, amount)
        
        # 设置过期时间(到周期结束)
        ttl = self._get_ttl(definition.period)
        await self.redis.expire(key, ttl)
        
        remaining = definition.limit - new_used if definition.limit > 0 else -1
        
        return QuotaUsage(
            resource=resource,
            used=new_used,
            limit=definition.limit,
            reset_at=self._get_reset_time(definition.period),
            remaining=remaining,
            exceeded=definition.limit > 0 and new_used > definition.limit
        )
    
    async def get_all_quotas(
        self,
        tenant_id: str,
        plan: str = "free"
    ) -> Dict[QuotaResource, QuotaUsage]:
        """获取所有资源配额情况"""
        quotas = {}
        for resource in QuotaResource:
            quotas[resource] = await self.get_quota_usage(tenant_id, resource, plan)
        return quotas
    
    def _get_reset_time(self, period: QuotaPeriod) -> datetime:
        """计算配额重置时间"""
        now = datetime.utcnow()
        
        if period == QuotaPeriod.MINUTE:
            return now + timedelta(minutes=1)
        elif period == QuotaPeriod.HOUR:
            return now + timedelta(hours=1)
        elif period == QuotaPeriod.DAY:
            return now + timedelta(days=1)
        elif period == QuotaPeriod.WEEK:
            return now + timedelta(weeks=1)
        elif period == QuotaPeriod.MONTH:
            # 下月同一天
            if now.month == 12:
                return now.replace(year=now.year + 1, month=1)
            else:
                return now.replace(month=now.month + 1)
        elif period == QuotaPeriod.YEAR:
            return now.replace(year=now.year + 1)
        
        return now
    
    def _get_ttl(self, period: QuotaPeriod) -> int:
        """获取周期对应的TTL(秒)"""
        now = datetime.utcnow()
        reset_time = self._get_reset_time(period)
        return int((reset_time - now).total_seconds())
```

# gateway/middleware/quota.py
from fastapi import Request, HTTPException
from starlette.middleware.base import BaseHTTPMiddleware
from starlette.types import ASGIApp

from core.quota import QuotaManager, QuotaResource

class QuotaMiddleware(BaseHTTPMiddleware):
    """配额检查中间件"""
    
    def __init__(self, app: ASGIApp, quota_manager: QuotaManager):
        super().__init__(app)
        self.quota_manager = quota_manager
    
    async def dispatch(self, request: Request, call_next):
        # 获取租户ID
        tenant_id = request.headers.get("X-Tenant-ID")
        if not tenant_id:
            return await call_next(request)
        
        # 根据路径确定资源类型
        resource = self._get_resource_type(request.url.path, request.method)
        
        if resource:
            # 检查配额
            allowed = await self.quota_manager.check_quota(
                tenant_id=tenant_id,
                resource=resource,
                amount=1
            )
            
            if not allowed:
                raise HTTPException(
                    status_code=429,
                    detail={
                        "error": "Quota exceeded",
                        "resource": resource.value,
                        "message": "Your quota for this resource has been exceeded. Please upgrade your plan."
                    }
                )
            
            # 执行请求
            response = await call_next(request)
            
            # 成功后消耗配额
            if response.status_code < 400:
                await self.quota_manager.consume_quota(
                    tenant_id=tenant_id,
                    resource=resource,
                    amount=1
                )
            
            return response
        
        return await call_next(request)
    
    def _get_resource_type(self, path: str, method: str) -> Optional[QuotaResource]:
        """根据请求路径确定资源类型"""
        if method == "POST":
            if "/applications" in path:
                return QuotaResource.APPLICATIONS
            elif "/conversations" in path:
                return QuotaResource.CONVERSATIONS
            elif "/messages" in path or "/chat" in path:
                return QuotaResource.MESSAGES
        
        if "/api/" in path:
            return QuotaResource.API_CALLS
        
        return None

from abc import ABC, abstractmethod

from pydantic import BaseModel

class BaseOAuth2Provider(ABC):
    """OAuth2 Provider抽象基类"""
    
    name: str
    authorize_url: str
    token_url: str
    userinfo_url: str
    scopes: list = ["openid", "email", "profile"]
    
    def __init__(self, client_id: str, client_secret: str, redirect_uri: str):
        self.client_id = client_id
        self.client_secret = client_secret
        self.redirect_uri = redirect_uri
    
    @abstractmethod
    def get_authorize_url(self, state: str, **kwargs) -> str:
        """获取授权URL"""
        pass
    
    @abstractmethod
    async def exchange_code(self, code: str) -> OAuth2Token:
        """交换授权码获取Token"""
        pass
    
    @abstractmethod
    async def get_userinfo(self, token: OAuth2Token) -> OAuth2UserInfo:
        """获取用户信息"""
        pass
    
    async def refresh_token(self, refresh_token: str) -> OAuth2Token:
        """刷新Token（可选实现）"""
        raise NotImplementedError()

# auth/providers/google.py
import httpx
from auth.providers.base import BaseOAuth2Provider, OAuth2Token, OAuth2UserInfo

class GoogleOAuth2Provider(BaseOAuth2Provider):
    """Google OAuth2 Provider"""
    
    name = "google"
    authorize_url = "https://accounts.google.com/o/oauth2/v2/auth"
    token_url = "https://oauth2.googleapis.com/token"
    userinfo_url = "https://www.googleapis.com/oauth2/v2/userinfo"
    scopes = ["openid", "email", "profile"]
    
    def get_authorize_url(self, state: str, **kwargs) -> str:
        from urllib.parse import urlencode
        params = {
            "client_id": self.client_id,
            "redirect_uri": self.redirect_uri,
            "response_type": "code",
            "scope": " ".join(self.scopes),
            "state": state,
            "access_type": "offline",
            "prompt": "consent"
        }
        return f"{self.authorize_url}?{urlencode(params)}"
    
    async def exchange_code(self, code: str) -> OAuth2Token:
        async with httpx.AsyncClient() as client:
            response = await client.post(
                self.token_url,
                data={
                    "code": code,
                    "client_id": self.client_id,
                    "client_secret": self.client_secret,
                    "redirect_uri": self.redirect_uri,
                    "grant_type": "authorization_code"
                }
            )
            response.raise_for_status()
            data = response.json()
            
            return OAuth2Token(
                access_token=data["access_token"],
                refresh_token=data.get("refresh_token"),
                token_type=data.get("token_type", "Bearer"),
                expires_in=data["expires_in"],
                scope=data.get("scope")
            )
    
    async def get_userinfo(self, token: OAuth2Token) -> OAuth2UserInfo:
        async with httpx.AsyncClient() as client:
            response = await client.get(
                self.userinfo_url,
                headers={"Authorization": f"Bearer {token.access_token}"}
            )
            response.raise_for_status()
            data = response.json()
            
            return OAuth2UserInfo(
                provider=self.name,
                provider_user_id=data["id"],
                email=data["email"],
                name=data.get("name"),
                picture=data.get("picture"),
                raw_data=data
            )

from typing import Type, Dict, Optional
from auth.providers.base import BaseOAuth2Provider

class ProviderRegistry:
    """Provider注册表"""
    
    _providers: Dict[str, Type[BaseOAuth2Provider]] = {}
    _configs: Dict[str, dict] = {}
    
    @classmethod
    def register(cls, name: str, provider_class: Type[BaseOAuth2Provider]):
        """注册Provider"""
        cls._providers[name] = provider_class
    
    @classmethod
    def configure(cls, name: str, config: dict):
        """配置Provider"""
        cls._configs[name] = config
    
    @classmethod
    def get_provider(cls, name: str) -> Optional[BaseOAuth2Provider]:
        """获取Provider实例"""
        if name not in cls._providers:
            return None
        
        config = cls._configs.get(name, {})
        provider_class = cls._providers[name]
        
        return provider_class(**config)
    
    @classmethod
    def list_providers(cls) -> list:
        """列出所有可用Provider"""
        return [
            {
                "name": name,
                "enabled": name in cls._configs
            }
            for name in cls._providers.keys()
        ]

class TokenPayload(BaseModel):

    roles: List[str]   # 角色列表

def create_access_token(
    user_id: str,
    tenant_id: str,
    roles: List[str],
    permissions: List[str],
    expires_delta: Optional[timedelta] = None
) -> str:
    """创建Access Token"""
    from core.config import settings
    
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=30)
    
    payload = {
        "sub": user_id,
        "tenant_id": tenant_id,
        "roles": roles,
        "permissions": permissions,
        "type": "access",
        "iat": datetime.utcnow(),
        "exp": expire,
        "jti": str(uuid.uuid4())
    }
    
    return jwt.encode(
        payload,
        settings.SECRET_KEY,
        algorithm="HS256"
    )

def create_refresh_token(
    user_id: str,
    expires_delta: Optional[timedelta] = None
) -> str:
    """创建Refresh Token"""
    from core.config import settings
    
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(days=7)
    
    payload = {
        "sub": user_id,
        "type": "refresh",
        "iat": datetime.utcnow(),
        "exp": expire,
        "jti": str(uuid.uuid4())
    }
    
    return jwt.encode(
        payload,
        settings.SECRET_KEY,
        algorithm="HS256"
    )

def decode_token(token: str) -> Optional[TokenPayload]:
    """解码Token"""
    from core.config import settings
    
    try:
        payload = jwt.decode(
            token,
            settings.SECRET_KEY,
            algorithms=["HS256"]
        )
        return TokenPayload(**payload)

# auth/token_blacklist.py
from redis.asyncio import Redis
from datetime import datetime, timedelta

class TokenBlacklist:
    """Token黑名单管理"""
    
    def __init__(self, redis: Redis):
        self.redis = redis
        self.blacklist_key = "token:blacklist"
    
    async def revoke_token(self, jti: str, expires_at: datetime):
        """吊销Token"""
        ttl = int((expires_at - datetime.utcnow()).total_seconds())
        
        if ttl > 0:
            await self.redis.setex(
                f"{self.blacklist_key}:{jti}",
                ttl,
                "revoked"
            )
    
    async def is_revoked(self, jti: str) -> bool:
        """检查Token是否已吊销"""
        exists = await self.redis.exists(f"{self.blacklist_key}:{jti}")
        return exists > 0

## 16. AgentScope 借鉴设计

> **设计来源**: AgentScope 框架核心架构模式

### 16.1 Environment 通信模型

AgentScope 采用**Agent-Environment-Agent**通信模式，作为多 Agent 的通信中枢：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Environment 通信模型                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   ┌──────────┐      ┌─────────────────────────────────┐      ┌──────────┐  │
│   │ Agent A  │─────▶│                                 │◀─────│ Agent B  │  │
│   └──────────┘      │                                 │      └──────────┘  │
│                     │      Environment                │                     │
│   ┌──────────┐      │      (通信中枢)                  │      ┌──────────┐  │
│   │ Agent C  │─────▶│                                 │◀─────│ Agent D  │  │
│   └──────────┘      │                                 │      └──────────┘  │
│                     │  • 消息路由                      │                     │
│                     │  • 状态同步                      │                     │
│                     │  • 任务协调                      │                     │
│                     └─────────────────────────────────┘                     │
└─────────────────────────────────────────────────────────────────────────────┘
```

**SmartLink 实现方案**:
- 使用 Redis Pub/Sub实现 Environment 通信
- 消息格式标准化 (sender_id, receiver_id, message_type, content, timestamp)
- 支持单播、广播、组播多种通信模式

### 16.2 Pipeline 编排模式

AgentScope 提供两种核心编排模式：

| 模式 | 描述 | 适用场景 | SmartLink 采纳 |
|------|------|----------|---------------|
| **SequentialPipeline** | 智能体按顺序执行，前一个输出作为后一个输入 | 简单线性流程 | ✅ 采纳 |
| **FanoutPipeline** | 多个智能体并行处理相同输入，结果聚合 | 并行任务处理 | ✅ 采纳 |

### 16.3 Agent 抽象层次

| Agent 类型 | 用途 | SmartLink 采纳 |
|-----------|------|---------------|
| **BaseAgent** | 所有 Agent 的父类，定义统一接口 (run/send/receive) | ✅ 采纳 |
| **ChatAgent** | 专注对话交互，内置多轮对话记忆管理 | ✅ 采纳 |
| **ToolAgent** | 聚焦工具调用，自动处理参数校验→调用→结果解析 | ✅ 采纳 |
| **CustomAgent** | 开发者重写_think() 和_act() 实现个性化逻辑 | ✅ 采纳 |

### 16.4 记忆系统设计

AgentScope 的记忆系统分为三层：

| 层级 | 存储 | 用途 | SmartLink 实现 |
|------|------|------|---------------|
| **L1 短期记忆** | 内存 (Redis) | 当前对话上下文 | Redis Session Store |
| **L2 长期记忆** | 向量库 (PgVector) | 历史交互、知识库 | PostgreSQL+PgVector |
| **L3 结构化记忆** | 关系数据库 | 用户偏好、任务状态 | PostgreSQL 关系表 |

### 16.5 工具封装模式

AgentScope 采用标准化 Tool 封装：
```python
class BaseTool(ABC):
    name: str
    description: str
    
    @abstractmethod
    async def _run(self, **kwargs) -> Any:
        pass
    
    async def run(self, **kwargs) -> Dict:
        # 1. 参数验证
        self.validate_params(kwargs)
        # 2. 权限检查
        await self.check_permission(kwargs)
        # 3. 执行工具
        try:
            result = await self._run(**kwargs)
            return {"success": True, "result": result}
        except Exception as e:
            return {"success": False, "error": str(e)}
```

## 17. 工作流编排设计

> **设计参考**: AgentScope Pipeline + CoPaw 工作流

### 17.1 工作流类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| **线性工作流** | 开始→意图识别→工具调用→结果处理→响应生成→结束 | 简单任务，步骤固定 |
| **分支工作流** | 根据意图识别结果路由到不同 Agent(订单/售后/人工) | 多意图路由，条件分支 |
| **并行工作流** | 多个 Agent 并行处理，结果聚合 | 多任务并行，结果聚合 |

### 17.2 工作流定义 Schema

采用 YAML 配置化定义：
```yaml
name: customer_service_workflow
version: "1.0"
nodes:
  - id: intent_recognition
    type: agent
    agent: ChatAgent
  - id: order_query
    type: agent
    agent: ToolAgent
    condition: "intent == 'order'"
edges:
  - from: intent_recognition
    to: order_query
    condition: "intent == 'order'"
```

### 17.3 Human-in-the-Loop 设计

参考 CoPaw 的人工介入机制：
- **审批请求**: Agent 执行敏感操作前请求人工审批
- **审批队列**: Redis 队列存储待审批请求
- **审批回调**: 人工审批后回调继续工作流

### 17.4 工作流状态机

| 状态 | 描述 | 可转换状态 |
|------|------|------------|
| PENDING | 等待执行 | RUNNING, CANCELLED |
| RUNNING | 执行中 | COMPLETED, FAILED, WAITING_APPROVAL, PAUSED |
| WAITING_APPROVAL | 等待审批 | RUNNING, CANCELLED |
| PAUSED | 已暂停 | RUNNING, CANCELLED |
| COMPLETED | 已完成 | - |
| FAILED | 失败 | - |
| CANCELLED | 已取消 | - |
